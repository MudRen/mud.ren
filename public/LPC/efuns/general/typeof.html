typeof(3)		MudOS v21c2		(5 Sep 1994)

	名称:
		typeof() - 传回一个式子 (expression) 的资料型态 (type).
	语法:
		int typeof( mixed var );

		整数 typeof( 混合 var );
	用法:
		传回一个式子的资料型态.  中定义传回值为:

		整数		INT		2
		字串		STRING		4
		阵列		ARRAY		8 
		物件		OBJECT		16
		映射		MAPPING		32
		函式指标	FUNCTION	64
		浮点数		FLOAT		128
		暂存区		BUFFER		256

	参考:
		allocate(3), allocate_mapping(3), strlen(3)
