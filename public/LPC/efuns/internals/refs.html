refs(3)			MudOS v21c2		(5 Sep 1994)

	名称:
		refs - 传回一个资料结构的参考数字.
	语法:
		int refs( mixed data );

		整数 refs( 混合 data );
	用法:
		refs() 会传回 data 的参考数字. 这个函式可以在传回 data
		以前, 判断是否要复制一份资料结构.
	参考:
		children(3), inherit_list(3), deep_inherit_list(3),
		objects(3)
