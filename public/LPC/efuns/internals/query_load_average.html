query_load_average(3)	MudOS v21c2		(5 Sep 1994)

	名称:
		query_load_average()  - 报告 mud 的系统平均负载量 (load
					average).
	语法:
		string query_load_average( void );

		字串 query_load_average( 无参数 );
	用法:
		这个函式传回一个字串, 报告两个统计数字:
		(1) 平均每秒钟的使用者命令数量.
		(2) 平均每秒钟编译的程式行数.
	参考:
		rusage(3)
	翻译:
		Spock @ FF	97.May.1.		(printed 3/16/95)
