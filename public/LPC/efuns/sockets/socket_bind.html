socket_bind(3)         MudOS (5 Sep 1994)          socket_bind(3)

     名称：
          socket_bind() - 命名(bind)一个 socket

     语法：
          #include 

          int socket_bind( int s, int port );

          整数 socket_bind( 整数 s, 整数 port );

     用法：
          socket_bind() 为一个未命名的 socket 指定一个名字。当一个
          socket 由 socket_create(3) 建立後，它已存在於一个命名空间
          （位址族, address family）但还没有指定名字。socket_bind()
          需要埠号来为 socket s 命名。

     传回值：
          socket_bind() 传回：

               成功时是 EESUCCESS。

               失败时是一个有如下意义的负值。

     错误值：
          EEFDRANGE      描述值(descriptor)超出□围。

          EEBADF         无效的描述值。

          EESECURITY     企图违反安全。

          EEISBOUND      socket 已命名。

          EEADDRINUSE    此位址已被使用。

          EEBIND         命名问题。

          EEGETSOCKNAME  getsockname 问题。

     参照：
          socket_connect(3), socket_create(3), socket_listen(3)
