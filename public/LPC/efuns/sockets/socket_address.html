socket_address(3)      MudOS (5 Sep 1994)       socket_address(3)

     名称：
          socket_address() - 传回一个 efun socket 的远端位址

     语法：
          #include 

          string socket_address( int s );

          字串 socket_address( 整数 s );

     用法：
          socket_address() 传回 efun socket s 的远端位址。传回的位
          址格式如下：

               "127.0.0.1 23".

     传回值：
          socket_address() 传回：

               成功时是一个字串形式的位址。

               失败时是一个空字串。

     参照：
          socket_connect(3), socket_create(3), resolve(3),
          query_host_name(3), query_ip_name(3), query_ip_number(3)
