ed_cmd(3)		MudOS v21c2		(5 Sep 1994)

	名称:
		ed_cmd() -  对一个正在执行的 ed 工作 (session)  
			    送出一个命令.
	语法:
		string ed_cmd( string cmd )

		字串 ed_cmd( 字串 cmd )
	用法:
		这个函式只有在 __OLD_ED__ 不予定义 (define) 时有效.

		命令 cmd 会被送到一个正在执行的 ed 工作 (session) , 
		并传回命令执行後输出的结果.
