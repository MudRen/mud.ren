in_edit(3)		MudOS v21c2		(5 Sep 1994)

	名称:
		in_edit() - 判断一个玩家是否在文书编辑程式中.
	语法:
		string in_edit( object default : F_THIS_OBJECT );

		字串 in_edit( 物件 预设值 : F_THIS_OBJECT );
	用法:
		如果一个指定的物件正在文书编辑工作中, 就传回被编辑
		档案名称. 如果不是, 就传回 0.
	参考:
		ed(3), in_input(3)
	翻译:
		Spock @ FF	97.Feb.14.		(printed 3/16/95)
