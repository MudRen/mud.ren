query_host_name(3)	MudOS v21c2		(5 Sep 1994)

	名称:
		query_host_name() - 传回使用的主机名称 (host name)
	语法:
		string query_host_name( void );

		字串 query_host_name( 无参数 );
	用法:
		query_host_name() 传回使用的主机名称.
	传回值:
		如果询问主机成功, 则传回一个字串.

		如果失败, 则传回一个空字串.
	参考:
		resolve(3), socket_address(3), query_ip_name(3),
		query_ip_number(3)
