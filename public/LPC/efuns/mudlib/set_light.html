set_light(3)		MudOS v21c2		(5 Sep 1994)

	名称:
		set_light() - 更改或询问一个物件的照明程度 (light level).
	语法:
		int set_light( int light_level_adjustment );

		整数 set_light( 整数 light_level_adjustment );
	用法:
		将 light_level_adjustment 指定为 0, 则询问物件目前的
		照明程度. 指定为正整数则增加照明的程度, 负整数降低.
	翻译:
