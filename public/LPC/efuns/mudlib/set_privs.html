set_privs(3)		MudOS v21c2		(5 Sep 1994)

	名称:
		set_privs() - 设定一个物件的私有字串 (privs string).
	语法:
		void set_privs( object ob, string privs );

		无传回值 set_privs( 物件 ob, 字串 privs );
	用法:
		设定 privs 为 ob 的私有字串.

		这个超越函式 (efun) 只有在驱动程式编译时定义 PRIVS
		才有效.
	参考:
		privs_file(4), query_privs(3)
	翻译:
		Spock @ FF	97.Jul.21.		(printed 3/16/95)
