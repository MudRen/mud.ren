字符串的子范围 (sub ranging) - 注解: Grey@TMI-2

您可以使用子字符串运算 (substring operation)　 从一个字符串变量中, 取出字符串的一部份 (substring). 其形式为:

    字符串名称[n1..n2]
正整数代表由左开始算起, 而负整数从左算起. 如果数字超过字符串的长度, 则视为字符串的长度.

如果两个数字相等［字符串名称[n1..n1]］, 则返回字符串的第 n1 个字符 (character). 如果两个数字都超过字符串的长度, 则会返回空字符串 (null string)［""］. 如果第一个数字所指的位置在第二个数字之后, 也会返回空字符串.

范例:

    str = "abcdefg"
则
    str[0..0] ==　　　　　　 "a"
    str[0..-1] ==　　　　　 "abcdefg"
    str[-4..-2] ==　　　　 "def"
    str[-7..6] ==　　　　　 "abcdefg"
    str[3..2] ==　　　　　　 ""
