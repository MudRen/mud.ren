array.2d

LPC 可以做到二元数组, 这很简单. 只要把它们当作一个数组的数组即可.
举例来说:
    a = allocate(10);
    a[0] = allocate(10);
    a[1] = allocate(10);
    ...以此类推...
然后您可以用以下的方式引用 (reference) 数组 0 , 元素 0 :
    a[0][0]
您不能声明 (declare)　 一个超过一元的数组 (如果您有形态检查 (type checking), 而使用 *　 形态的符号) , 但是您还是可以有一个一元以上的数组. 如果您有形态检查, 您可能会把数组声明成混合 (mixed)　 形态.
　
可以使用以下这种方法:
    mixed a;
    a = ({ ({ 1, 2, 3 }), ({ 1, 2, 3 }) });
在上面的这个范例中, a[0] 是 ({ 1, 2, 3 }), 而 a[0][2] 是 3.

或这种方法:

    mixed a;
    a = ({ 0, 0, 0, 0 });　 /* 设定数组大小为 4 */
    a[0] = ({ 1, 2, 3 });
    a[1] = ({ 1, 2, 3 });
    ...以此类推...
