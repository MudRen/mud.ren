if

* if 与 else 语句:

LPC 的 if 语句与 C 提供的相同. 语法如下:

    if (运算式)
      语句;
第二种语法:
    if (运算式) {
      语句;
    }
第三种:
    if (运算式一) {
      　语句一;
    } else {
      　语句二;
    }
第四种:
    if (运算式一) {
      语句一;
    } else if (运算式二) {
      运算式二;
    }
else if 子句没有什么数量上的限制.

另一个好用的程序结构是 ? : 运算符. 它与 C 提供的一样. 语法如下:

　　　 运算式一 ? 运算式一为真则进行运算式二 : 运算式一为假则进行运算式三

某些情形下, ? : 算是以下程序结构的缩写:

　　　 if (运算式一)
　　　　　　　 变量 = 运算式二;
　　　 else
　　　　　　　 变量 = 运算式三;

上面这段程序可以转换成下面这行:

　　　 变量 = 运算式一 ? 运算式二 : 运算式三;
