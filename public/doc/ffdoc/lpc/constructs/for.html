for

* LPC 的 for 循环 :

LPC 的 for 循环与 C 提供的相同. 语法如下:

    for ( 运算式 0; 运算式 1; 运算式 2 ) {
      语句;
      ...;
    }
执行循环之前, 会先计算「运算式 0」. 每次循环周期开始时, 计算「运算式 1」. 如果「运算式 1」计算的结果得到 0, 则循环停止. 「计算式 2」则于每次循环周期结束时计算之.

循环里面的 break 会让循环停止. continue 则会跳过此次循环周期剩余的部分, 从下一个周期开始 (会先计算「运算式 2」).

典型的 for 循环, 用以固定的次数重覆执行一段循环中的语句.

范例:

    int i;
    for (i = 0; i < 10; i++) {
      write("i == " + i + "\n");
      write("10 - i == " + (10 - i) + "\n");
    }
