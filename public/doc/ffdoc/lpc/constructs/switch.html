* switch 语句

LPC 的 switch 语句与 C　 的 switch 语句几乎一样. 唯一不同的地方是 LPC　 的 switch　 判断依据的运算式可以使用字符串或整数.
语法如下:

    switch (运算式) {
      case 常数一 : 语句一;
        break;
      case 常数二 :　 语句二;
        break;
      default :　 语句三;
        break;
    }
switch 可以代替一堆 if else if else if else 结构. 上面的 switch 等于:
    tmp = 运算式;

    if (tmp == 常数一) {

      语句一;
      ...;
    } else if (tmp == 常数二) {
      语句二;
      ...;
    } else {
      语句三;
      ...;
    }
switch　 与 if 语句之间最大的不同处, 在于某个 case 最后如果没有 break; 语句, 则会执行下一个 case.
