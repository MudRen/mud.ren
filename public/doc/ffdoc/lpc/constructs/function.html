* LPC 的函数 (function 或称之为 method) :

LPC 的函数与 C 提供的相似, 但是有些差异 (与 ANSI C 提供的最为相似).
语法如下:

    返回值的类型 函数名称( 参数一的类型 参数一, 参数二的类型 参数二, ... )
    {
      声明变量;
      ...;

      程序语句;
      ...;

      return 返回值变量;
    }
请注意, 「返回值变量」必须符合返回值的类型.

如果函数不必返回一个值, 则声明返回值类型为 void (无返回值). 举例来说:

    void 函数名称( 参数一的类型 参数一, ... )
    {
      语句;
      ...;
    }
下面此行调用上面的函数:
    函数名称( 参数一, 参数二, 参数三, ... );
您可以调用其他物件中的函数, 方法如下:
    物件->函数名称( 参数一, 参数二, 参数三, ... );
或是:
    call_other( 物件, 函数名称, 参数一, 参数二, ... );
