save_variable(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 save_variable() - 将变量值储存为一个字符串 (string).
　语法:
    　 string save_variable( mixed var );

    　 字符串 save_variable( 混合 var );

　用法:
    　 把变量值储存成一个字符串. 储存变量的格式跟 save/restore_object 函数一样.
　参考:
    　 save_object(3)
　翻译:
    　 Spock @ FF 97.Jul.4.　 (printed 3/16/95)
