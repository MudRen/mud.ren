write_buffer(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 write_buffer() - 将一个缓冲区 (buffer) 写入一个文件, 或从某个数据来源读进一个缓冲区.
　语法:
    　 int write_buffer( string | buffer dest, int start, mixed source );

    　 整数 write_buffer( 字符串 或 缓冲区 dest, 整数 start, 混合 source );

　用法:
    　 如果 dest 是一个文件, 则 source 必须为一个整数 (并以 network-byte-order　 写入文件) 、一个缓冲区、或一个字符串, 而从文件 dest 的第 start　 字节开始写入 source 的内容.

    　 如果 dest 是一个缓冲区, 则 source 会从第 start　 字节开始写入缓冲区. 如果 source 是整数, 则会以 network-byte-order　 的格式写入.

　参考:
    　 read_buffer(3), allocate_buffer(3)
　作者:
    　 Truilkan
