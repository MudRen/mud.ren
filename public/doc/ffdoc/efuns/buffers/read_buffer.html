read_buffer(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 read_buffer() - 以缓冲区数据 (buffer) 的类型返回文件的内容, 或是以字符串类型 (string) 返回缓冲区一部份的数据.
　语法:
    　 string | buffer read_buffer( string | buffer src, int start,　 int len );

    　 字符串 或 缓冲区 read_buffer( 字符串 或 缓冲区 src, 整数 start, 整数 len );

　用法:
    　 如果 src　 是一个字符串 (文件名称) , 就会从 start　 指定的第几字节 (byte) 开始, 读取 len　 指定长度的文件内容. 读取的内 容以缓冲区的数据类型返回. 如果不指定 start 和 len , 则会读取整个文件的内容.

    　 如果 src　 是一个缓冲区, 就从 start　 指定的第几位元开始读取 len 长度的内容. 然后以字符串的类型返回.

    　 请注意, 你可以从一个文件读进缓冲区的数据, 受到编译时段组态文件 (runtime config file)　 中定义的 maximum byte transfer (最大字节传输量) 控制.

　参考:
    　 write_buffer(3), allocate_buffer(3), bufferp(3), read_bytes(3), write_bytes(3)
