restore_object(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 restore_object() - 从一个文件读回一个物件的变量值.
　语法:
    　 int restore_object( string name, int flag );

    　 整数 restore_object( 字符串 name, 整数 flag );

　用法:
    　 从叫作 name 的文件读回目前物件的变量值. 如果 flag 是 1, 则非静态变量 (non-static) 不会在读回之前被清除为零. (普通状况下, 非静态变量都是零).

    　 如果读回的过程出错, 则所有的变量都不会更动, 并返回错误.

　参考:
    　 save_object(3)
