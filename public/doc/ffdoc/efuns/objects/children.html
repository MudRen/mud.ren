children(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 children() - 返回一个指定物件 (object) 所复制出来的物件数组.
　语法:
    　 object * children( string name );

    　 物件 *children( 字符串 name );

　用法:
    　 这个函数返回一个物件数组 (array), 物件群则是文件名称为 name 的物件所复制出来的. 一个使用的范例是找出所有从使用者物件 (user object)　 复制出来的物件:
    　
      　　 object *list;

      　　 list = children("/obj/user");

    　 这样可以让您找出所有的使用者 (包括已经断线的使用者),　 而 users() 只会回报还能互动的 (interactive) 使用者.
　参考:
    　 deep_inherit_list(3), inherit_list(3), objects(3)
　翻译:
    　 Spock @ FF 97.Jan.26.　 (printed 3/16/95)
