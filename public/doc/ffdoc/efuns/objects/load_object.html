load_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 load_object() - 以文件名称来寻找或载入一个物件
　语法:
    　 object load_object( string str );

    　 物件 load_object( 字符串 str );

　用法:
    　 用文件名称 str　 寻找物件. 如果文件存在而且物件尚未载入, 则载入文件. 否则此函数就返回 0.
　参考:
    　 file_name(3), stat(3), find_object(3)
　翻译:
    　 Spock @ FF 97.Feb.18.　 (printed 3/16/95)
