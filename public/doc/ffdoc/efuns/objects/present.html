present(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 present() - 以物品的辨识名称 (id) 寻找物件.
　语法:
    　 object present( mixed str, object ob );

    　 物件 present( 混合 str, 物件 ob );

　用法:
    　 如果一个物件的名字为 str, 就返回此物件.

    　 str 也可以是一个物件, 这样会比较坑邙简单.

    　 如果指定 ob, present() 会寻找 ob 的内容物 (inventory), 不然会寻找目前物件的内容物, 以及目前物件所处环境的内容物.

　参考:
    　 move_object(3), environment(3)
　翻译:
