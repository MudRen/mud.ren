tell_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 tell_object() - 将消息传递给一个物件.
　语法:
    　 void tell_object( object ob, string str );

    　 无返回值 tell_object( 物件 ob, 字符串 str );

　用法:
    　 将消息 str　 传递给物件 ob. 如果 ob 是一个可互动的 (interactive) 物件 (玩家), 此段消息就对 ob 显示. 反之, 则会传递给 ob 的函数 catch_tell().
　参考:
    　 message(3), write(3), shout(3), say(3), tell_room(3)
　翻译:
    　 Spock @ FF 97.jul.24.　 (printed 3/16/95)
