query_heart_beat(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 query_heart_beat() - 查询一个物件的心跳 (heart beat) 状况.
　语法:
    　 int query_heart_beat( object );

    　 整数 query_heart_beat( 物件 );

　用法:
    　 对于已经调用过 set_heart_beat() 函数的物件, query_heart_beat() 返回其心跳值. 如果不指定 ob, 则预设为目前的物件. 没有心跳的物件则返回 0.
　参考:
    　 heart_beat(4), set_heart_beat(3)
　翻译:
    　 Spock @ FF 97.Apr.30.　 (printed 3/16/95)
