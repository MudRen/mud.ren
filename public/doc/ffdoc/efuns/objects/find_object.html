find_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 find_object() - 以指定的文件名称寻找一个物件.
　语法:
    　 object find_object( string str );

    　 物件 find_object( 字符串 str );

　用法:
    　 用文件名称 str　 寻找一个物件. 如果要寻找的物件是一个复制出来的物件, 亦可以用 file_name()　 返回的文件名称作为参数来寻找.
　参考:
    　 file_name(3), stat(3)
