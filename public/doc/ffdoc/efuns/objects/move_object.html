move_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 move_object() - 把目前的物件搬到其他的环境 (environment) 中.
　语法:
    　 void move_object( object item, mixed dest );

    　 无返回值 move_object( 物件 item, 混合 dest );

　用法:
    　 把物件 item 搬到物件 dest 里面. item 必须是 this_object(). 如果 item 是 this_object(), move_object() 可以只指定 dest 一个参数.
　参考:
    　 this_object(3), move(4)
　翻译:
    　 Spock @ FF 97.Apr.15.　 (printed 3/16/95)
