file_name(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 file_name() - 取得一个物件的文件名称.
　语法:
    　 string file_name( object ob );

    　 字符串 file_name( 物件 ob );

　用法:
    　 file_name() 返回载入 ob 时所用的文件名称. 如果物件是复制出来的, 则 file_name()　 返回的名称并不是实际上磁碟中所用的名称, 而是在原来的文件名称之后加上「#」 号, 再加上物件的复制编号. 当游戏启动时, 物件的复制编号从 0　 开始起算, 每复制一个相同的物件就增加一号. 所以每一个复制出来的物件都有自己独特的编号. 如果没有指定 ob, 则 ob 预设为 this_object().
　翻译:
