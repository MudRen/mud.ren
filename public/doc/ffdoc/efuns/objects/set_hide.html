set_hide(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 set_hide() - 对一个可以藏匿的物件设定藏匿标志 (hide flag).
　语法:
    　 void set_hide( int flag );

    　 无返回值 set_hide( 整数 flag );

　用法:
    　 以整数 flag 设定一个物件的藏匿标志. 标志只能是 0　 或 1, 分别代表取消和开始藏匿状态. 只有在 master()->valid_hide(ob) 对此物体返回真值时, 此物体才能藏匿. 当物体处于藏匿状态时, 只有其他有能力藏匿的物体才能使用 find_object()　 或返回多个物件的函数如 users(), children(), all_inventory() 等等找到此物体.
　参考:
    　 valid_hide(4)
　翻译:
