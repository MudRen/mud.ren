objects(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 objects - 返回一个已载入之物件的数组 (array).
　语法:
    　 object *objects( string func, object ob );
    　 object *objects( function f );

    　 物件 *objects( 字符串 func, 物件 ob );
    　 物件 *objects( 函数指针 f );

　用法:
    　 objects() 返回 mud　 里已经载入之物件的数组. 请注意, 如果系统设定的最大数组长度 (maximum array size)太低, objects() 返回的数组内容会中断在最大长度的地方. 这样一来, objects() 也就没有什么用处.

    　 如果指定了 func 和 ob 两个参数, 则会调用 ob->func(), 并以所有已经载入的物件作为 func() 的参数. 如果函数返回非零值, 此物件就会被 objects()　 返回来. 如果是零则不会.

    　 如果指定 f, 所有已载入的物件都会调用函数指针 f. 例如: objects( (: clonep :) ) 会返回现存全部的复制物件的列表.

　参考:
    　 livings(3), users(3)
