set_heart_beat(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 set_heart_beat() - 开始或取消一个物件的心跳 (heartbeat).
　语法:
    　 int set_heart_beat( int flag );

    　 整数 set_heart_beat( 整数 flag );

　用法:
    　 参数指定为 0　 取消物件的心跳.
    　 参数指定为 1　 会使这个物件每一次心跳调用 heart_beat() 函数一次.　 (每一次心跳的秒数通常是两秒钟, 但是 mud　 的管理者可以定义不同的秒数) 参数指定为大于 1　 的数字, 则这个物件每隔多少次心跳才调用一次 heart_beat(). 需要注意的是, 您的 mud　 管理者可能将系统设定为大于 1 的数字视为 1.
　参考:
    　 heart_beat(4), query_heart_beat(3)
　翻译:
    　 Spock @ FF 97.Jul.21.　 (printed 3/16/95)
