save_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 save_object() - 将一个物件的变量值储存到一个文件中.
　语法:
    　 int save_object( string name, int flag );

    　 整数 save_object( 字符串 name, 整数 flag );

　用法:
    　 把这个物件所有的非静态变量 (non-static variables) 储存到名叫 name 的文件中. 这个储存的动作由主控物件 (master object) 中的 valid_write() 函数检查是否允许这个物件写入. 如果第二个参数指定为 1, 则变量值为零的的变量也会储存进去 (一般说来不会是零). 而物件类型的变量则会储存为 0.
　返回值:
    　 save_object() 成功会返回 1, 失败就返回 0.
　参考:
    　 restore_object(3)
