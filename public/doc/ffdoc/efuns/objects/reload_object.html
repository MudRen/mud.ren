reload_object(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 reload_object - 把一个物件还原成刚刚载入的状态.
　语法:
    　 void reload_object( object ob );

    　 无返回值 reload_object( 物件 ob );

　用法:
    　 对 ob 调用 reload_object() 时, 所有由驱动程序管理的物件特性 (心跳 heart_beat,　 外部调用 call_outs,　 光线 light, 阴影 shadow 等) 和内部变量都会重新初始化 (re-initialized), 然后调用　 create(). 这个函数的效果相当于先摧毁再载入物件, 用不着存取磁碟.
　参考:
    　 export_uid(3), new(3), clone_object(3), destruct(3)
　翻译:
    　 Spock @ FF 97.May.26.　 (printed 3/16/95)
