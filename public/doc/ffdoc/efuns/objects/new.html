new(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 new() - 载入并复制一个物件.
　语法:
    　 object new( string name );

    　 物件 new( 字符串 name );

　用法;
    　 以 name 定义的文件名称载入一个新的物件, 并且给此物件一个独特的名称. 此函数返回这个新的物件. 一个 environment() 为非零值的物件无法复制.
　参考:
    　 clone_object(3), destruct(3), move_object(3)
　翻译:
    　 Spock @ FF 97.Apr.24.　 (printed 3/16/95)
