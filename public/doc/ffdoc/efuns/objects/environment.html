environment(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 environment() - 返回一个物件所身处的环境.
　语法:
    　 object environment( object ob );

    　 物件 environment( 物件 ob );

　用法:
    　 返回含有物件 ob 的环境物件. 如果没有指定参数 (argument), ob 预设为 this_object().
　参考:
    　 this_object(3)
　翻译:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
