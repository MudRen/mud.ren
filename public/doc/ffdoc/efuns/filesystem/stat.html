stat(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 stat() - 返回与一个文件或一个目录相关的信息.
　语法:
    　 mixed stat( string str, int default: 0 );

    　 混合 stat( 字符串 str, 整数 预设值: 0 );

　用法:
    　 如果 str 是一个普通文件的名字 (不是目录), 则 stat() 会返回一个数组. 此数组含有关于此文件的信息. 数组的格式如下:

    　 ({ file_size, last_time_file_touched, time_object_loaded })
     

    　 ({ 文件名称, 最后一次的更动日期, time_object_loaded })

    　 如果 str　 是一个目录, 或是第二个参数为 -1 , 则 stat() 与 get_dir(3) 的作用相同.

　参考:
    　 get_dir(3), file_size(3)
