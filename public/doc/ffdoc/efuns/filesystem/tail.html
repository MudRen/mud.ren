tail(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 tail() - 显示一个文件的最后一部份.
　语法:
    　 int tail( string path );

    　 整数 tail( 字符串 path );

　用法:
    　 此外部函数 (efun) 显示文件 path 的最后一部份. 成功则返回 1, 不成功返回 0. (例如文件读取权限不够)
　参考:
    　 read_file(3), read_buffer(3), file_size(3)
　翻译:
    　 Spock @ FF 97.Jul.24.　 (printed 3/16/95)
