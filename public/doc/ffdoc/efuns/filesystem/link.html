link(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 link() - 将一个文件与另一个文件链结
　语法:
    　 void link( string original, string reference );

    　 无返回值 link( 字符串 original, 字符串 reference );

　用法:
    　 将文件 reference　 链结到文件 original . link()　 会使主控物件 (master object)　 调用 valid_link( original, reference). 如果 valid_link() 返回 0, 则这次的 link() 失败. 如果 valid_link() 返回 1, 则还要由 rename() 以同样的参数 (argument) 调用, 判断是否成功. 如果 rename(), valid_link()　 均返回 1, 则 link() 成功.

    　 注意: link() 函数创造一个强链结 (hard link) , 而非符号链结 (symbolic link).

　参考:
    　 rm(3), rmdir(3), rename(3), mkdir(3), cp(3)
