write_file(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 write_file() - appends a string to a file
　语法:
    　 int write_file( string file, string str, int flag );

    　 整数 write_file( 字符串 file, 字符串 str, 整数 flag );

　用法:
    　 将字符串 str　 加到文件 file 中. 成功则返回 1, 失败返回 0. 如果 flag 为 1, 则 write_file() 会覆写 (overwrite) file 为 str, 而非加入 str.
　参考:
    　 read_file(3), write_buffer(3), file_size(3)
