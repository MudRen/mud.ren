write_bytes(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 write_bytes() - 将连续的字节写入文件.
　语法:
    　 int write_bytes( string path, int start, string series );

    　 整数 write_bytes( 字符串 path, 整数 start, 字符串 series );

　用法:
    　 write_bytes() 以连续的字节, 从文件 path 的第 start 字节开始写入. 如果失败则返回 0, 成功返回 1.
　参考:
    　 write_file(3), read_bytes(3)
