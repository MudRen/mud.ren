file_size(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 file_size() - 取得文件的大小.
　语法:
    　 int file_size( string file );

    　 整数 file_size( 字符串 file );

　用法:
    　 file_size() 返回文件 file 的位元数. 返回 -1 代表 file 不存在, 或是您的权限无法读写此文件. 返回 -2 则代表 file　 是一个目录.
　参考:
    　 stat(3), get_dir(3)
　翻译:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
