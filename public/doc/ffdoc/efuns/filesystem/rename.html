rename(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 rename() - 更改一个文件的名称.
　语法:
    　 int rename( string src, string dst );

    　 整数 rename( 字符串 src, 字符串 dst );

　用法:
    　 把叫做 src 的文件改名叫做 dst.
　返回值:
    　 改名成功就返回 0.　 失败则返回非零值.
　参考:
    　 rm(3), rmdir(3), cp(3), link(3)
　翻译:
    　 Spock @ FF 97.May.27.　 (printed 3/16/95)
