cp(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 cp() - 复制一个文件
　语法:
    　 int cp( string src, string dst );

    　 整数 cp( 字符串 src, 字符串 dst );

　用法:
    　 复制文件 src 为文件 dst.
　返回值:
    　 如果复制成功就返回 1, 失败则返回负值.
　参考:
    　 rm(3), rmdir(3), rename(3), link(3)
　翻译:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
