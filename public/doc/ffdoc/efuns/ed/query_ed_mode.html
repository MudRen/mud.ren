query_ed_mode(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 query_ed_mode() - 查询目前编辑程序 (ed) 执行的状态.
　语法:
    　 int query_ed_mode()

    　 整数 query_ed_mode()

　用法:
    　 寻问目前此物件执行编辑程序 (ed) 的状态. query_ed_mode() 的返回值如下:
    　
    　 0　 - 此物件正在普通的 ed 提示符号 ( ':', 冒号) 下.
    　 -1 - 此物件并没有进入 ed 里
    　 -2 - 此物件正处于 ed 求助说明第一页的 more 提示符号下.
    　 >0 - 此物件正在编辑一行内容, 这个数字就是第几行.
　翻译:
