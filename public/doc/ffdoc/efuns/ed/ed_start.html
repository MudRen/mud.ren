ed_start(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 ed_start() - 开始执行一个 ed 工作.
　语法:
    　 string ed_start( string file | void, int restricted | void )

    　 字符串 ed_start( 字符串 file 或 无参数, 整数 restricted 或 无参数 )

　用法:
    　 这个函数只有 __OLD_ED__ 不予定义 (define) 时有效.

    　 这个函数开始执行内部的文书编辑程序, 可以指定或不指定要予以编辑的文件 file.　 函数会返回执行完 ed 的编辑结果. 当 ed 程序仍在执行时, 可以使用 ed_cmd() 执行 ed 的命令.

    　 如果 restricted 的值为 1　 , 所有更改文件的命令都会被视为无效.

    　 一个物件同一时间内只能执行一个 ed 工作.

　翻译:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
