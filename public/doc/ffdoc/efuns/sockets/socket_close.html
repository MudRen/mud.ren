socket_close(3)　 MudOS v21c2　 (5 Sep 1994)

　名称：
    　 socket_close() - 关闭一个 socket
　语法：
    　 #include <socket_err.h>

    　 int socket_close( int s );

    　 整数 socket_close( 整数 s );

　用法：
    　 socket_close() 会关闭 socket s。这会释放出一个被使用的 socket efun 插槽(slot)。
　返回值：
    　 socket_close() 返回：
     
      　　 成功时是 EESUCCESS。

      　　 失败时是含如下意义的负值。

　错误值：
    　 EEFDRANGE　　　　　 描述值(descriptor)超出范围。

    　 EEBADF　　　　　　　　 无效的描述值。

    　 EESECURITY　　　　 企图违反安全。

　参照：
    　 socket_accept(3), socket_create(3)
