socket_address(3)　 MudOS v21c2　 (5 Sep 1994)

　名称：
    　 socket_address() - 返回一个 efun socket 的远端位址
　语法：
    　 #include <socket_err.h>

    　 string socket_address( int s );

    　 字符串 socket_address( 整数 s );

　用法：
    　 socket_address() 返回 efun socket s 的远端位址。返回的位址格式如下：
     
      　　 "127.0.0.1 23".
　返回值：
    　 socket_address() 返回：
     
      　　 成功时是一个字符串形式的位址。

      　　 失败时是一个空字符串。

　参照：
    　 socket_connect(3), socket_create(3), resolve(3), query_host_name(3), query_ip_name(3), query_ip_number(3)
