socket_listen(3)　 MudOS v21c2　　 (5 Sep 1994)

　名称：
    　 socket_listen() - 倾听(listen)一个 socket 的连接(connection)。
　语法：
    　 #include <socket_err.h>

    　 int socket_listen( int s, string listen_callback );

    　 整数 socket_listen( 整数 s, 字符串 listen_callback );

　用法：
    　 为了要接受连接，首先要以 socket_create(3) 建立一个 socket，再以 socket_listen(3) 将这个 socket 放到倾听模式，最后才以 socket_accept(3) 接受这个连接。socket_listen() 调用只能作用于 STREAM 或是 MUD 类型的 sockets。

    　 listen_callback 参数是给驱动程序在得到正倾听中 socket 的连接时调用的函数名称。这个倾听回呼(callback)函数应该是这个格式：
     

      　　 void listen_callback(int fd)

    　 这里的 fd 是正倾听中的 socket。

　返回值：
    　 socket_listen() 返回：
     
      　　 成功时是 EESUCCESS。

      　　 失败时是一个含有如下意义的负值。

　错误码：
    　 EEFDRANGE　　　　　 描述值(descriptor)超出范围。

    　 EEBADF　　　　　　　　 无效的描述值。

    　 EESECURITY　　　　 企图违反安全。

    　 EEMODENOTSUPP　 不支援此 socket 类型。

    　 EENOADDR　　　　　　 socket 尚未设定位址。

    　 EEISCONN　　　　　　 socket 已连接。

    　 EELISTEN　　　　　　 倾听问题。

　参照：
    　 socket_accept(3), socket_connect(3), socket_create(3)
