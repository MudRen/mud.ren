socket_error(3)　 MudOS v21c2　　 (5 Sep 1994)

　名称：
    　 socket_error() - 返回一个 socket 错误码的文字语句
　语法：
    　 #include <socket_err.h>

    　 string socket_error( int error );

    　 字符串 socket_error( 整数 error );

　用法：
    　 socket_error() 返回一个描述所产生之错误的字符串。
　返回值：
    　 socket_error() 返回：
     
      　 成功时是一个描述错误码的字符串。

      　 错误的输入值时是 "socket_error: invalid error number"。
      　　　　　　　　　　　　　　 （译注："socket_error: 无效的错误码"）

　参照：
    　 socket_create(3), socket_connect(3)
