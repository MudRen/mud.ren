filter_array(3)　　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 filter_array() - 返回一个过滤旧数组内容的新数组.
　语法:
    　 mixed *filter_array( mixed *arr, string fun, object ob, mixed extra, ... );
    　 mixed *filter_array( mixed *arr, function f, mixed extra, ... );

    　 混合 *filter_array( 混合 *arr, 字符串 fun, 物件 ob, 混合 extra, ... );
    　 混合 *filter_array( 混合 *arr, 函数 f, 混合 extra, ... );

　用法:
    　 filter_array()　 返回一个数组, 这个数组里保留 arr 经过物件 ob 中的函数 fun　 过滤之后的结果. 每一个在数组 arr 中的元素 (element)　 会调用函数 fun　 一次, 并且当成 fun 的参数 (parameter). 如果指定参数 extra　 与之后的参数, 则在每一次调用函数 fun　 时当成第二、第三个参数....一个物件在 fun　 返回 1　 之后, 才视为成功地通过 fun　 的过滤 (之后物件也包含在返回的数组中) . 在第二种语法中, 如果指定的参数是函数 f, 就与第一种语法中的 ob->fun()　 的用法相同. 如果 arr　 不是一个数组, filter_array()　 返回 0.
　参考:
    　 sort_array(3), map_array(3)
　翻译:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
