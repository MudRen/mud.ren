member_array(3)　 MudOS v21c2　 (5 Sep a994)

　名称:
    　 member_array() -　 在一个数组 (array)　 或字符串 (string) 中找寻指定的项目 (item) , 并返回此项目为第几项的索引.
　语法:
    　 int member_array( mixed item, mixed * | string arr, void | int start );

    　 整数 member_array( 混合 item, 混合 * 或 字符串 arr, 无参数 或 整数 start );
     

　用法:
    　 返回 item 在数组 arr　 中第一次出现时, 出现在第几项. 如果使用 start 参数, 则从 start 指定的第几项之后找起. 如果找不到 item, 则返回 -1.
