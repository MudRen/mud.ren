allocate(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 allocate() - 配置内存给一个数组 (array).
　语法:
    　 mixed *allocate( int size );

    　 混合 *allocate( 整数 size );

　用法:
    　 配置内存给一个有 size 个元素 (element)　 的数组. 元素的数目必须大于 0, 而且不可以超过系统设定的上限 (通常约 10000 左右) . 所有元素的初始值 (initialized) 均为 0.
　参考:
    　 sizeof(3), allocate_mapping(3)
　翻译:
    　 Spock @ FF 96.Oct.12.　 (printed 3/16/95)
