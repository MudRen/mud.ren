apply(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 apply() - 处理函数指针 (function pointer).
　语法:
    　 mixed apply( mixed f, mixed *args )

    　 混合 apply( 混合 f, 混合 *args )

　用法:
    　 如果 f 是一个函数, 就会调用 f 并用数组 args 当作 f 的参数 (argument).
     
　 范例:
    　　 apply( f, ({ 1, 2, 3 }) ); 与 evaluate( f, 1, 2, 3 ); 是相同的.
