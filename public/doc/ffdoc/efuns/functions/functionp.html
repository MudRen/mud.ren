functionp(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 functionp() - 判断指定的变量是否为一个函数指针 (function pointer). 如果是函数指针, 则再判断为何种类型.
　语法:
    　 int functionp( mixed arg );

    　 整数 functionp( 混合 arg );

　用法:
    　 如果 arg 是一个函数指针则返回非零值 (nonzero), 如果不是则返回 0.
    　 函数指针是一种「函数」(function) 类型的指针变量.
    　 以下是一个范例:
    　
      　 f = (: obj, func :);
    　 返回的值指示出函数指针的类型, 而这些返回值被定义在驱动程序中的 "include/function.h" 文件中.

    　 function pointer type　 value
    　 ---------------------------------------------
    　 call_other　　 FP_CALL_OTHER
    　 lfun　　　 FP_LOCAL
    　 efun　　　　　FP_EFUN
    　 simul_efun　　 FP_SIMUL
    　 functional　　 FP_FUNCTIONAL

    　 另外在某些情形下, 会加上以下的值:

    　 (有参数)　　 FP_HAS_ARGUMENTS
    　 (创造函数的物件被摧毁)　 FP_OWNER_DESTED
    　 (函数无法连结)　　 FP_NOT_BINDABLE

    　 最后三个值为位元值 (bit value), 可以用位元测试.
    　 FP_MASK 可以忽略这些位元值, 测试基本的函数指针类型
    　

　 范例:
    　 测试一个函数指针变量是否为外部函数指针:
    　
      　 if ((functionp(f) & FP_MASK) == FP_EFUN) ...
    　 测试其是否有参数:
    　
      　 if (functionp(f) & FP_HAS_ARGUMENTS) ...
