previous_object(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 previous_object() - 返回调用目前这个函数的物件 (或物件列表).
　语法:
    　 mixed previous_object( int | void );

    　 混合 previous_object( 整数 或 无参数 );

　用法:
    　 返回一个调用目前函数之物件的物件指针. 请注意, 局部函数 (local function)　 的调用不会让 previous_object()　 设定为目前的物件, 不会改变 previous_object()　 返回的物件. 如果指定一个正整数为参数, 就返回先前第几个调用的物件. previous_object(0) 如同 previous_object(), previous_object(1) 是 previous object 的 previous_object(), 以此类推.　 previous_object(-1) 则返回所有的 previous object 数组.
　参考:
    　 call_other(3), call_out(3), origin(3)
　翻译:
    　 Spock @ FF 97.Apr.24.　 (printed 3/16/95)
