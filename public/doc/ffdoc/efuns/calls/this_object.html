this_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 this_object() - 返回调用此函数物件的物件指针 (object pointer).
　语法:
    　 object this_object( void );

    　 物件 this_object( 无参数 );

　用法:
    　 返回这个物件的物件指针. 如果想调用同一个物件却非同一个源代码文件的函数时 (即调用所继承 (inherite) 物件的函数), 可用 this_object()　 达到目的.
　参考:
    　 this_player(3), previous_object(3), origin(3)
　翻译:
    　 Spock @ FF 97.Jul.25.　 (printed 3/16/95)
