shadowp(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 shadowp() - 判断指定的物件是否为其他物件的投影 (shadow).
　语法:
    　 object shadowp( object ob );

    　 物件 shadowp( 物件 ob );

　用法:
    　 返回投影 ob 的物件, 如果 ob 不是任何物件的投影, 则返回 0.
　参考:
    　 shadow(3), objectp(3), query_shadowing(3), valid_shadow(4)
　翻译:
    　 Spock @ FF 97.Jul.25.　 (printed 3/16/95)
　译注:
    　 此份 query_shadowing 文件, 在 MudOS v21c2 所附文件中即为 shadowp 之内容. 原文附件中, 无 shadowp 之文件.
