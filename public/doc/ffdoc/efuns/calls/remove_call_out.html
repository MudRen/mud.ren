remove_call_out(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 remove_call_out() - 取消一个延迟中的外部调用 (call_out).
　语法:
    　 int remove_call_out( string fun );

    　 整数 remove_call_out( 字符串 fun );

　用法:
    　 取消目前这个物件延迟调用的外部调用函数 fun. 返回值就是外部调用还剩下多少延迟时间. 如果函数 fun　 已经没有延迟的外部调用, 就返回 -1.
　参考:
    　 call_out(3), call_out_info(3)
　翻译:
    　 Spock @ FF 97.May.27.　 (printed 3/16/95)
