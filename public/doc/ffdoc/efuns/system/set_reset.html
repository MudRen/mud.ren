set_reset(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 set_reset - 更改一个物件距离下次重新设定 (reset) 的时间.
　语法:
    　 varargs void set_reset( object ob, int time );

    　 不定参数 无返回值 set_reset( 物件 ob, 整数 time );

　用法:
    　 设定物件 ob 距离下次重新设定的时间为整数 time 指定的秒数. 如果不指定 time, 则使用驱动程序普通设定的公式.
     
　 公式:
    　　 reset time = current_time + reset_time / 2 + random(reset_time / 2)
　参考:
    　 reset(4)
　翻译:
