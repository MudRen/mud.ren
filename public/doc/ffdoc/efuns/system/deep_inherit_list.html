deep_inherit_list(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 deep_inherit_list() - 取得一个物件所继承的父物件. (parent object)
　语法:
    　 string *deep_inherit_list( object obj );

    　 字符串 *deep_inherit_list( 物件 obj );

　用法:
    　 返回一个物件所继承之所有父物件的文件名称, 包括直接继承与间接继承的父物件.
　参考:
    　 inherit_list(3), inherits(3)
　翻译:
    　 Spock @ FF 97.Feb.04.　 (printed 3/16/95)
