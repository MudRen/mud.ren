errorp(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 errorp() - 判断一个指定的变量是否为错误码 (error code).
　语法:
    　 int errorp( mixed arg );

    　 整数 errorp( 混合 arg );

　用法:
    　 如果参数 arg　 是一个整数值的错误码, errorp() 返回 1 .

    　 您可能会修改外部函数 (emulated functions, efun), 而能返回一个标准的错误码. 所以程序可以写成像下面的样子:
     

      　 if (errorp(result = efun_call()))
        　　 printf("error = %d\n", result);

    　 在以后的计划中, 我们会加入一个新的函数 perror(result) 用来返回整数值的错误代码与一个错误消息的字符串.

　参考:
    　 nullp(3), undefinedp(3)
