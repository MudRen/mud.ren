error(3)　 MudOS v21c2　 error(3)

　名称:
    　 error - 产生一个执行时段 (run_time) 错误.
　语法:
    　 void error( string err );

    　 无返回值 error( 字符串 err );

　用法:
    　 调用 error() 时, 会产生一个执行时段错误 err. 并中止目前正在执行的执行绪脉络 (thread). 而发生错误的程序会被追踪错误的出处, 并纪录在除错纪录 (debug log) 的文件中.
　参考:
    　 catch(3), throw(3), error_handler(4)
