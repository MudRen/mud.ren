ctime(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 ctime() - 返回一个时间字符串
　语法:
    　 string ctime( int clock );

    　 字符串 ctime( 整数 clock );

　用法:
    　 返回一个日期和时间的字符串, 而参数 clock 是自从 1970 年算起的秒数.
　参考:
    　 time(3), time_expression(3), localtime(3), uptime(3)
　翻译:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
