time(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 time() - 返回现在时刻距离 1970 年一月一日有多少秒.
　语法:
    　 int time( void );

    　 整数 time ( 无参数 );

　用法:
    　 返回从 1970 年一月一日午夜 (格林威治时间) 开始到现在, 总共过了多少秒.
　参考:
    　 ctime(3), localtime(3), time_expression(3), uptime(3)
　翻译:
    　 Spock @ FF 97.Jul.25.　 (printed 3/16/95)
