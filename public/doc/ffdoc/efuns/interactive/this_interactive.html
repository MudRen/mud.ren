this_interactive(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 this_interactive - 返回调用此函数的玩家物件.
　语法:
    　 object this_interactive();

    　 物件 this_interactive();

　用法:
    　 返回调用此函数的玩家物件. 基本上 this_interactive() 会返回 this_player(), 而不像 this_player() 会受到 enable_commands() 或 command()　 的影响.
　参考:
    　 this_object(3), this_player()
　翻译:
    　 Spock @ FF 97.Jul.24.　 (printed 3/16/95)
