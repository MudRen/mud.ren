exec(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 exec() -　 将一个可互动的 (interactive) 玩家 (player) 物件切换连结 (connection) 到另外一个物件上.
　语法:
    　 int exec( object to, object from );

    　 整数 exec( 物件 to, 物件 from );

　用法:
    　 这个函数允许指定的物件把可互动的连结 (link) 迁移到另外一个物件上. 也就是说, 成功地执行 exec( to, from ) 之后, interactive( to ) 会返回 1, 而 interactive( from ) 会返回 0. 而控制 from 的玩家会转移控制权到 to 的身上.

    　 请注意这个函数的权力非常大, 不当地使用会危及 MUD　 的系统安全. 正确地限制使用此函数的方法是另外写一个同名的模拟外部函数 (simulated emulated function, simul_efun), 并使用 valid_override(4)　 来限制此一模拟外部函数的使用. (意即 efun::exec() )

    　 exec() 函数在成功地转换之后返回 1, 转换失败则返回 0.

　参考:
    　 interactive(3), valid_override(4)
