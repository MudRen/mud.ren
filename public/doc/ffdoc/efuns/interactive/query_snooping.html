query_snooping(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 query_snooping() - 返回正在监听 (snoop) 的物件.
　语法:
    　 object query_snooping( object ob );

    　 物件 query_snooping( 物件 ob );

　用法:
    　 如果互动 (interactive)　 物件 ob 正在监听另一个互动物件, 则返回被监听的物件. 如果 ob 没有监听其他物件, 则返回 0.
　参考:
    　 snoop(3), query_snoop(3)
　翻译:
    　 Spock @ FF 97.May.1　 (printed 3/16/95)
