shout(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 shout() - 对所有活着的 (living) 物件送出消息.
　语法:
    　 void shout( string str );

    　 无返回值 shout( 字符串 str );

　用法:
    　 除了 this_player() 以外, 对所有活着的物件送出字符串 str.
　参考:
    　 message(3), write(3), tell_object(3), tell_room(3), say(3)
　翻译:
    　 Spock @ FF 97.Jul.23.　 (printed 3/16/95)
