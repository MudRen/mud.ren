disable_commands(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 disable_commands() - 设定一个活着的 (living) 物件为「非活着」(non-living)
　语法:
    　 int disable_commands( void );

    　 整数 disable_commands( 无参数 );

　用法:
    　 让一个活着的物件变成「非活着」(non-living). 所谓「非活着」, 就是让物件变成以下的状态:

    　 add_actions 失效.
    　 livingp 函数返回「伪」值 (false, 0).
    　 如果此物件是「可互动的」 (interactive), 就让这个使用者
    　 无法输入命令 (input_to 函数例外).

    　 disable_commands 永远返回 0 值.

　参考:
    　 enable_commands(3)
