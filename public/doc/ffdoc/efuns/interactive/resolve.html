resolve(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 resolve() - 转换一个网际网路地址 (internat address) 为网域名称 (domain name), 反之亦然..
　语法:
    　 int resolve( string address, string callback_func );

    　 整数 resolve( 字符串 address, 字符串 callback_func );

　用法:
    　 resolve() 会转换像 "140.128.136.11" 形式的 address　 成为 "bbs.csmc.edu.tw" 这种形式的网域名字. 或是把 "bbs.csmc.edu.tw" 这种形式的 address　 转换成 "140.128.136.11"　 形式的地址. 转换成功后, 会调用目前物件的 callback_func. callback_func 的格式为:

    　 void callback(string address, string resolved, int key);

    　 无返回值 callback( 字符串 address, 字符串 resolved, 整数 key );

    　 key 是 resolve() 的返回值. address 是转换后的地址或网域. 如果转换失败, 则会返回 0.

　参考:
    　 query_host_name(3), socket_address(3), query_ip_name(3), query_ip_number(3)
