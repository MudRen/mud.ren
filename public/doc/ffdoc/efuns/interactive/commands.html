commands(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 commands() - 返回一些关于此使用者可以使用的命令信息
　语法:
    　 mixed *commands( void );

    　 混合 *commands( 无参数 );

　用法:
    　 返回一个包含四项数据的数组 (array), 说明 this_object() 可以使用的动作 (action) . 第一项数据是命令名称 ( 即 add_action()　 的命令) ; 而第二项是命令的标志 (flags) (即 add_action() 的第三个参数, 预设值为 0　 );　 第三项数据是定义此动作的物件. 第四项数据是被此动作调用的函数 (function) (此项数据如果是个函数指针 [function pointer] 则返回 "<function>").
　参考:
    　 add_action(3), enable_commands(3), disable_commands(3)
　翻译:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
