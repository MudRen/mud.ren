remove_action(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 remove_action - 从局部函数 (local function) 中解除一个命令动词 (command verb)
　语法:
    　 int remove_action( string fun, string cmd );

    　 整数 remove_action( 字符串 fun, 字符串 cmd );

　用法:
    　 remove_action(3) 从一个物件的局部函数中解除一个命令动词.
    　 基本上, remove_action() 是相对于 add_action(3) 和 add_verb(3) 的函数. 当您不需要一个动词时, 就可以用 remove_action() 解除.
　返回值:
    　 remove_action() 的返回值:

    　 1 解除动词成功.

    　 0 解除失败.

　参考:
    　 add_action(3), query_verb(3), init(4)
　翻译:
    　 Spock @ FF 97.May.27.　 (printed 3/16/95)
