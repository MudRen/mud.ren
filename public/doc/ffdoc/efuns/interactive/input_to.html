input_to(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 input_to() - 让下一行输入的内容送往一个指定的函数
　语法:
    　 varargs void input_to( string | function fun, int flag, ... );

    　 不定参数 无返回值 input_to( 字符串 或 函数指针 fun, 整数 flag, ... );

　用法:
    　 让使用者下一行输入的内容作为函数指针 fun 的参数. 输入的内容不会被驱动程序 (driver) 分析 (parse).

    　 请注意, input_to　 是「非阻隔性」的, 意即调用 input_to 的物件不会为此暂停下来等待输入. 物件会继续执行 input_to　 之后的任何程序语句. 在使用者尚未输入以前, 不会调用函数指针 fun.

    　 如果同一个物件调用 input_to() 超过一次以上, 只有第一次调用有效.

    　 如果参数 flag 为非零值, 则玩家输入的内容不会回应 (echo) 在画面上, 也不会出现在监听 (snoop)　 的画面上 (在输入密码时很有用).

    　 使用者输入的一行内容会被当成调用函数指针 fun　 时的第一个参数 (类型为字符串).　 在 flag 之后的参数会当成 fun　 的第二个、第三个......等等的参数.

　参考:
    　 call_other(3), call_out(3), get_char(3)
