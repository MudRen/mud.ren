receive(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 receive() - 对目前的物件显示一段消息.
　语法:
    　 int receive( string message );

    　 整数 receive( 字符串 message );

　用法:
    　 这个外部函数 (efun) 是作为 mud　 驱动程序 (driver) 的 add_message() 函数的介面. 这个函数设计的目的是对目前物件显示消息. 如果目前的物件是「可互动的」则返回 1. 不是则返回 0. receive() 应被 catch_tell(4) 或是 receive_message(4) 调用.
　参考:
    　 catch_tell(4), receive_message(4), message(3)
　翻译:
    　 Spock @ FF 97.May.24.　 (printed 3/16/95)
