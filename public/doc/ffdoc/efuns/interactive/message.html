message(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 message() - 送消息给活着的 (living) 物件.
　语法:
    　 void message( mixed class, string message, mixed target, mixed exclude );

    　 无返回值 message( 混合 class, 字符串 message, 混合 target, 混合 exclude );

　用法:
    　 message() 调用目标名单中所有物件 (除了 exclude 名单以外) 的 receive_message( 混合 class, 字符串 message ) 函数. 这样基本上只是告诉此物件一个消息.
    　
    　 class 是此段消息的种类, 用于处理消息的介面之类的地方. 例如 combat, shout, emergency 等等.

    　 message 是欲送出的消息字符串.

    　 target 是欲送达消息的物件名单, 可以是一个物件或物件指针 (pointer). 如果目标是一个非活着的 (non-living) 的物件, 在此物件内的所有物件都会收到消息.

    　 exclude 是不应收到此消息的物件名单, 可以是一个物件或物件数组 (array).

　参考:
    　 say(3), write(3), shout(3), tell_object(3), tell_room(3)
　翻译:
