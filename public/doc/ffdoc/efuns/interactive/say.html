say(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 say() - 把消息传递给所有在同一个环境中的使用者.
　语法:
    　 varargs void say( string str, object obj );

    　 不定参数 无返回值 say( 字符串 str, 物件 obj );

　用法:
    　 把消息传递给位于跟发出消息者同一环境的物体, 以及所有在发出消息者里面的物体. 发出消息者只能是 this_player(), 或是　 this_player() == 0 的 this_object().

    　 第二个参数可有可无. 如果指定了第二个参数 obj, 则消息不会传递给 obj. 如果 obj 不是一个物件, 而是一个物件数组 (an array of objects) , 则消息就不会传递给数组内的物件.

　参考:
    　 message(3), write(3), shout(3), tell_object(3), tell_room(3)
　翻译:
    　 Spock @ FF 97.Jul.4.　 (printed 3/16/95)
