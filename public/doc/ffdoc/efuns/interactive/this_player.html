this_player(3)　 MudOS v21c2　 (5 Sep 1994)

　语法:
    　 object this_player( int flag );

    　 物件 this_player( 整数 flag );

　用法:
    　 返回调用此函数的玩家 (player) 物件. 请注意, 即使是从玩家物件内部使用 this_player(), 其返回值也可能与 this_object() 不同. 如果以 this_player(1) 调用, 则返回调用此函数的可互动 (interactive)　 物件. this_player(1)　 的返回值在某些情形下与 this_player() 不同. 例如: 系统管理员 (admin)　 使用 command()　 强制一名玩家做出一些动作时.
　参考:
    　 this_object(3)
　翻译:
    　 Spock @ FF 97.Jul.25.　 (printed 3/16/95)
