query_ip_name(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 query_ip_name() - 返回指定玩家的 ip 名称.
　语法:
    　 string query_ip_name( object ob );

    　 字符串 query_ip_name( 物件 ob );

　用法:
    　 返回玩家 ob 的 IP 位址. 一个叫做 addr_server 的非同步程序会找出这些名称. 如果找不到 ip 名称, 就会返回 ip 号码.
　参考:
    　 query_ip_number(3), query_host_name(3), resolve(3), socket_address(3)
　翻译:
    　 Spock @ FF 97.May.1.　 (printed 3/16/95)
