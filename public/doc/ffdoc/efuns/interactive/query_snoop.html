query_snoop(3)　 MudOS v21c2 (5 Sep 1994)

　名称:
    　 query_snoop() - 返回谁正在监听指定的互动 (interactive) 物件.
　语法:
    　 object query_snoop( object ob );

    　 物件 query_snoop( 物件 ob );

　用法:
    　 如果互动物件 ob 正被另一个互动物件监听, 就返回监听者物件. 如果没有监听者, 则返回 0.
　参考:
    　 snoop(3), query_snooping(3)
　翻译:
    　 Spock @ FF 97.May.1.　 (printed 3/16/95)
