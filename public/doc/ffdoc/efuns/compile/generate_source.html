generate_source(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 generate_source() - 产生与指定物件相对照的 C 源代码
　语法:
    　 void generate_source( string file, void | string out_file );

    　 无返回值 generate_source( 字符串 file, 无参数 或 字符串 out_file );

　用法:
    　 generate_source() 调用 LPC -> C 的编译程序, 产生指定物件的原始码. 如果没有指定输出的文件, 则输出的结果会放在 SAVE_BINARIES　 的目录.
　参考:
    　 valid_asm(4), valid_compile_to_c(4)
　翻译:
    　 Spock @ FF 97.Feb.13.　 (printed 3/16/95)
