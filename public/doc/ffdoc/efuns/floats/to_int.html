to_int(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 to_int - 将一个浮点数或缓冲区 (buffer) 转换为一个整数.
　语法:
    　 int to_int( float | buffer x );

    　 整数 to_int( 浮点数 或 缓冲区 x );

　用法:
    　 如果 x　 是一个浮点数, 则 to_int() 返回 x　 小数点之前的整数值. 如果 x　 是一个缓冲区, 则 to_int() 返回缓冲区之中的整数 ( 以 network-byte-order 的形式 ).
　参考:
    　 to_float(3), read_buffer(3), sprintf(3), sscanf(3)
