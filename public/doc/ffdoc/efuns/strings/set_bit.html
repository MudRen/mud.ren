set_bit(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 set_bit() - 设定一个位元字符串 (bitstring) 中的一个位元 (bit).
　语法:
    　 string set_bit( string str, int n );

    　 字符串 set_bit( 字符串 str, 整数 n );

　用法:
    　 返回一个新的字符串, 新字符串将 str 中的第 n 位元设定为 1. 请注意, 原来的字符串 str 并没有更动内容.

    　 n 的最大值有其上限. 如果您想知道这个上限值, 请向机器的管理者查询.

    　 如果有必要, 新的字符串会自动加长.

    　 在可以印出的字符串中, 每一个字节 (byte) 含有 6　 个位元.

　参考:
    　 clear_bit(3), test_bit(3)
