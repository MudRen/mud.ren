implode(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 implode() - 连结多个字符串.
　语法:
    　 string implode( mixed *arr, string del );

    　 字符串 implode( 混合 *arr, 字符串 del );

　用法:
    　 把数组 arr 中可以找到的所有字符串连结在一起, 每一个字符串之间以字符串 del 隔开. 这个函数只会使用数组中的字符串. 数组中不是字符串的元素 (element) 会予以忽略.
　参考:
    　 explode(3), sprintf(3)
　翻译:
    　 Spock @ FF 97.Feb.14.　 (printed 3/16/95)
