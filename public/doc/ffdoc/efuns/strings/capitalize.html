capitalize(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 capitalize() - 把一个字符串的第一个英文字符 (character) 转成大写.
　语法:
    　 string capitalize( string str );

    　 字符串 capitalize( 字符串 str );

　用法:
    　 把传入字符串 str 的第一个英文字符转成大写, 并返回一个转换后的字符串.
　参考:
    　 lower_case(3)
　翻译:
    　 Spock @ FF 96.Oct.16.　 (printed 3/16/95)
