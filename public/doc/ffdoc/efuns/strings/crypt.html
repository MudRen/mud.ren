crypt(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 crypt() - 对一个字符串进行编码.
　语法:
    　 string crypt( string str, string seed );

    　 字符串 crypt( 字符串 str, 字符串 seed );

　用法:
    　 以字符串 seed 的前两个字符当作种子数, 对字符串 str　 进行编码. 如果 seed 为 0　 , 就会用随机数字进行编码.

    　 返回字符串的前两个字母即为种子数.

　翻译:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
