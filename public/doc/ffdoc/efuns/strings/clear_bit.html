clear_bit(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 clear_bit() - 清除一个字符串中的某一个 bit .
　语法:
    　 string clear_bit( string str, int n );

    　 字符串 clear_bit( 字符串 str, 整数 n );

　用法:
    　 返回一个新的字符串 (string) , 而新字符串中第 n　 个 bit 清除为 0. 请注意, 原来的字符串 str　 并没有更动内容.
　参考:
    　 set_bit(3), test_bit(3)
　翻译:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
