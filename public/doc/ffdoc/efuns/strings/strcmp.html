strcmp(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 strcmp() - 判断两个字符串间的语句关系 (lexical relationship).
　语法:
    　 int strcmp( string one, string two );

    　 整数 strcmp( 字符串 one, 字符串 two );

　用法:
    　 此数的 strcmp() 与 C　 函数库的 strcmp 相同. 如果字符串 one 语句上在 two　 之前 (即 one　 在英文字典中的顺序在 two 之前),　 则 strcmp() 返回比 0　 小的值. 如果两个字符串相同, 则返回 0. 如果 two　 语句顺序在 one　 之前, 则 strcmp()　 返回比 0　 大的值. 这个外部函数 (efun) 在使用 sort_array(3)　 排序时, 特别有用.
　参考:
    　 sort_array(3)
