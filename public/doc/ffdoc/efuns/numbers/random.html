random(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 random() - 返回一个「假性」乱数 (pseudo-random number).
　语法:
    　 int random( int n );

    　 整数 random( 整数 n );

　用法:
    　 返回一个范围从 0　 到 n - 1　 的假性乱数.
　翻译:
    　 Spock @ FF 97.May.2.　 (printed 3/16/95)
　译按:
    　 真正的乱数必须先给予一个种子数 (seed number), 再藉由查乱数表取得一组乱数, 而这些乱数值均介于 0 与 1　 之间.
