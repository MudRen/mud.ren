seteuid(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 seteuid() - 设定一个物件的有效使用者识别名称 (effective user id, euid)
　语法:
    　 int seteuid( string str );

    　 整数 seteuid( 字符串 str );

　用法:
    　 设定有效使用者识别名称为 str　 的内容. 在 master.c 中的 valid_seteuid() 函数决定着一个物件的有效使用者识别名称可以设定为何.

    　 当一个物件的有效使用者识别名称为 0, 则这个物件的使用者识别名称 (uid)　 只能用 export_uid() 更改.

    　 同时, 这个物件不能载入或是复制任何物件.

　参考:
    　 export_uid(3), getuid(3), geteuid(3), this_object(3), valid_seteuid(4)
