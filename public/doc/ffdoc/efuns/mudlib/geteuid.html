geteuid(3) MudOS v21c2 (5 Sep 1994) geteuid(3)

　名称:
    　 geteuid() - 返回一个物件或是一个函数的有效使用者识别名称 (effective user id, euid)
　语法:
    　 string geteuid( object | function );

    　 字符串 geteuid( 物件 或 函数指针 );

　用法:
    　 如果参数是一个物件, geteuid 会返回物件的有效使用者识别名称 (euid) . 如果指定的参数是一个函数指针 (function pointer,　 变量类型为 function), 则会返回创造这个函数指针之物件的 euid . 如果这个函数指针的变量, 在此物件创造时没有 euid,　 则会使用物件的使用者识别名称 (user id, uid) 代替.
　参考:
    　 seteuid(3), getuid(3), functionp(3), export_uid(3), previous_object(3), this_object(3), valid_seteuid(4)
