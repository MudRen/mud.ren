set_living_name(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 set_living_name() - 设定一个活着的 (living) 物件的生物名称 (living name).
　语法:
    　 void set_living_name( string name );

    　 无返回值 set_living_name( 字符串 name );

　用法:
    　 对一个活着的物件设定其生物名称. 设定之后, 此物件就可以用 find_living() 找到.
　参考:
    　 enable_commands(3), find_living(3), find_player(3)
　翻译:
    　 Spock @ FF 97.Jul.21.　 (printed 3/16/95)
