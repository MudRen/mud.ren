export_uid(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 export_uid() -　 设定一个物件的使用者识别名称 (user id, uid).
　用法:
    　 int export_uid( object ob );

    　 整数 export_uid( 物件 ob );

　用法:
    　 设定 ob 的使用者识别名称 (uid) 为 this_object() 的有效使用者识别名称 (effective user id, effective uid). 这个函数只有在 ob 的有效使用者识别名称为 0 时有效.
　参考:
    　 this_object(3), seteuid(3), getuid(3), geteuid(3), previous_object(3), valid_seteuid(4)
　翻译:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
