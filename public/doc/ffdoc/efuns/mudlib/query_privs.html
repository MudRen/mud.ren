query_privs(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 query_privs() - 返回一个物件的私有字符串 (privs string).
　语法:
    　 string query_privs( object ob );

    　 字符串 query_privs( 物件 ob );

　用法:
    　 返回一个物件的私有字符串. 此私有字符串由主控物件 (master object) 于编译程序时, privs_file()　 函数决定. 而此字符串可由 set_privs()　 函数更改.

    　 query_privs() 只有在驱动程序 (driver) 编译时, 定义 PRIVS 才有效.

　参考:
    　 privs_file(4), set_privs(3)
　翻译:
    　 Spock @ FF 97.Jul.25.　 (printed 3/16/95)
