dump_file_descriptors(3) MudOS v21c2　 (5 Sep 1994)

　名称：
    　 dump_file_descriptors() - 倾印 MudOS 程序的文件描述值表
　语法：
    　 void dump_file_descriptors( void );

    　 无返回值 dump_file_descriptors( 无参数 );

　用法：
    　 提供这个函数是为了帮助 MudOS 驱动程序的除错，和克服在一些未提供等效或是出众除错机制之 UN*X 作业系统在执行上的不足之处。这个函数的输出结果是和系统十分相关的。每个文件描述值都会被检查是否参考到一个已开启的文件。如果是的话，将会显示由系统调用 fstat() 得到的 stat 结构信息。
