mud_status(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 mud_status() -　 报告一些驱动程序 (driver) 及 mud 函数库 (mudlib)　 的统计数字.
　语法:
    　 void mud_status( int extra );

    　 无返回值 mud_status( 整数 extra );

　用法:
    　 这个函数把一些驱动程序及 mud　 函数库的统计数字显示在调用这个函数的人的萤幕上. 如果参数 extra　 为非零值, 则会显示更多的附加数据. 这个函数取代了 vanilla 3.1.2 版本中的 status 及 status tables 指令.
　参考:
    　 debug_info(3), dumpallobj(3), memory_info(3), uptime(3)
　翻译:
    　 Spock @ FF 97.Apr.15.　 (printed 3/16/95)
