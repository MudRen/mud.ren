debug_info(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 debug_info() - 显示除错 (debug) 信息
　语法:
    　 mixed debug_info( int operation, ... );
    　 mixed debug_info( 0, object ob );
    　 mixed debug_info( 1, object ob );

    　 混合 debug_info( 整数 operation, ... );
    　 混合 debug_info( 0, 物件 ob );
    　 混合 debug_info( 1, 物件 ob );

　用法:
    　 debug_info()　 是一个普通用途的措施, 用来为 MudOS 驱动程序除错. 而所需要的除错信息则由第一个参数决定. 其余的参数则由第一个参数的动作 (operation)　 决定.

    　 存在的动作 (0 与 1) 需要第二个物件参数, 而可能用来显示 MudOS 物件结构的不同范围 ( various fields of the MudOS object structure).

    　 以下的 LPC 码用来产生一个简单的输出结果:

    　 程序一:

      　 /* di0.c */
        　　 create() {
          　　　 debug_info(0, this_object());
        　　 }
    　 结果一:
        O_HEART_BEAT　　　　　 : FALSE
        O_IS_WIZARD　　　　　　 : FALSE
        O_ENABLE_COMMANDS : FALSE
        O_CLONE　　　　　　　　　　 : FALSE
        O_DESTRUCTED　　　　　 : FALSE
        O_SWAPPED　　　　　　　　 : FALSE
        O_ONCE_INTERACTIVE: FALSE
        O_RESET_STATE　　　　 : FALSE
        O_WILL_CLEAN_UP　　 : FALSE
        O_WILL_RESET: TRUE
        total light : 0
        next_reset　 : 720300560
        time_of_ref : 720299416
        ref　　　　　　　　 : 2
        swap_num　　　 : -1
        name　　　　　　　 : 'u/c/cynosure/di0'
        next_all　　　 : OBJ(bin/dev/_update)
        This object is the head of the object list.

    　 程序二:

      　 /* di1.c */
        　　 create() {
          　　　 debug_info(1, this_object());
        　　 }
    　 结果二:
        program ref's 1
        Name u/c/cynosure/di1.c
        program size 10
        num func's 1 (16)
        num strings 0
        num vars 0 (0)
        num inherits 0 (0)
        total size 104
