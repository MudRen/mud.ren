dump_prog(3)　 MudOS v21c2　　 (5 Sep 1994)

　名称：
    　 dump_prog() - 倾印/反组译一个 LPC 物件
    　
　 语法：
    　 void dump_prog( object ob, int flags default: 0, string file );

    　 无返回值 dump_prog( 物件 ob, 整数 flags 预设值: 0, 字符串 file );
    　

　 用法：
    　 dump_prog() 倾印关于程序 obj 的信息到某个文件 `file'，若没有指定 `file' 的话，则内定是 "/PROG_DUMP"。若调用这个函数的物件没有这个文件之写入权限的话，它将会失败。

    　 flags 可以是下列数值的组合：

        1 - 包含 i-code 的反组译；
        2 - 包含行号信息。
    　
　 参考：
    　 debug_info(3), dumpallobj(3)
