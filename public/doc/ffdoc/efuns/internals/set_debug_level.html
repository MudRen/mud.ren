set_debug_level(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 set_debug_level() - 设定驱动程序 (driver) 使用 debug() 宏 (macro) 时, 其除错 (debug) 的等级.
　语法:
    　 void set_debug_level( int level );

    　 无返回值 set_debug_level( 整数 level );

　用法:
    　 这个外部函数 (emulated function, efun) 只有在编译驱动程序时使用 -DDEBUG_MACRO 才有效. set_debug_macro() 让 mud　 执行时, 可以从内部控制除错消息的量及种类.

    　 请参考驱动程序原始码中的 debug.h, 其中有详细的定义.
