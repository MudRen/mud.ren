dump_socket_status(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 dump_socket_status() - 列出每一个 LPC 的 socket 状态
　语法:
    　 void dump_socket_status( void );

    　 无返回值 dump_socket_socket( 无参数 );

　用法:
    　 dump_socket_status()　 是一个诊断的功能, 列出 MudOS　 驱动程序组态之所有 LPC socket 的目前状态. 此函数常用于 LPC socket　 应用程序 (application)　 的除错. 显示的每一行各对照一个 LPC socket.　 第一列是 LPC socket 的第一个描述值 (descriptor), 第二列是第二个描述值, 以此类推. socket　 的总数在编译驱动程序时已经固定.

    　 第一列 Fd 是与 LPC socket 相关的作业系统的文件描述值. State 是目前的 LPC socket 运作情形. Mode　 是 socket 模式 (mode) , 是 socket_create()　 的参数之一. Local 和 Remote address　 是 Internet 位址和连接埠 (port number), 以 Internet 的分点表示法 (dot notation, 即 aaa.bbb.ccc.ddd 的格式) 显示. * 表示一个位址或是该处的值为零. 请注意, CLOSED　 状态的 LPC socket 为尚未使用的 socket, 所以该 socket　 显示的数据可能会特异.

    　 以下的输出数据由 Portals mud　 产生, 当时唯一的 socket 应用程序 (application)　 为 MWHOD. 数据显示目前正在使用两个 socket.　 一个是 STREAM 模式的 socket 正在倾听连线要求. 另一个是 DATAGRAM 模式的 socket 正在等待数据输入.
     
    Fd State Mode Local Address Remote Address
    13 LISTEN STREAM *.6889 *.*
    14 BOUND DATAGRAM *.6888 *.*
    -1 CLOSED MUD *.* *.*
    -1 CLOSED MUD *.* *.*
