dumpallobj(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 dumpallobj() - 报告所有载入之物件的一些统计数据.
　语法:
    　 void dumpallobj( string | void );

    　 无返回值 dumpallobj( 字符串 或 无参数 );

　用法:
    　 这个函数会印出所有已经载入之物件列表. 如果没有指定参数, 这些信息会倾印在 /OBJ_DUMP　 这个文件里. 如果有指定参数, 则参数就是指定要输出的文件名称.
　参考:
    　 mud_status(3), debug_info(3)
