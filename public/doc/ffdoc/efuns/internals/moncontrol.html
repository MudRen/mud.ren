moncontrol(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 moncontrol() - 开启或关闭侧写 (profiling).
　语法:
    　 void moncontrol( int on );

    　 无返回值 moncontrol( 整数 on );

　用法:
    　 如果参数 on 为 1, moncontrol()　 就开启侧写. 如果 on 指定为 0　 , moncontrol()　 会关闭侧写. 执行驱动程序 (driver) 时可以随时开关侧写, 典型的用法是只想侧写某些部份执行的情形. 如果驱动程序编译 (compile) 时, 没有开启侧写功能, moncontrol() 就没有效果.
　参考:
    　 opcprof(3), function_profile(3)
