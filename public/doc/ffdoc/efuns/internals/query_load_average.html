query_load_average(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 query_load_average()　 - 报告 mud 的系统平均负载量 (load average).
　语法:
    　 string query_load_average( void );

    　 字符串 query_load_average( 无参数 );

　用法:
    　 这个函数返回一个字符串, 报告两个统计数字:
      　 (1) 平均每秒钟的使用者命令数量.
      　 (2) 平均每秒钟编译的程序行数.
　参考:
    　 rusage(3)
　翻译:
    　 Spock @ FF 97.May.1.　 (printed 3/16/95)
