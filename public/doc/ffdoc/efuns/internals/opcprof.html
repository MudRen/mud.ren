opcprof(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 opcprof() - 报告一些外部函数 (efuns) 被调用的频率.
　语法:
    　 void opcprof( string | void );

    　 无返回值 opcprof( 字符串 或 无参数 );

　用法:
    　 这个函数印出每一个外部函数 (efunction)　 和超越操作子 (eoperator) 的统计数字列表. 如果没有指定参数, 所有的数据都会倾印到 /OPCPROF.efun 和 /OPCPROF.eoper　 这两个文件里. 如果指定了一个字符串, 则此字符串就用作倾印的文件名称.
　参考:
    　 function_profile(3)
　翻译:
