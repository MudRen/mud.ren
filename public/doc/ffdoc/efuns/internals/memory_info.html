memory_info(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 memory_info - 取得一个物件或整个 mud 的内存使用量.
　语法:
    　 varargs int memory_info( object ob );

    　 不定参数 整数 memory_info( 物件 ob );

　用法:
    　 如果指定 ob,　 则 memory_info()　 返回 ob 正在使用的内存量. 如果没有指定参数, 则返回整个 mud　 的内存使用量. 请注意, 整个 mud　 的内存使用量不是实际上系统配置给 mud 的内存量.
　参考:
    　 debug_info(3), malloc_status(3), mud_status(3)
