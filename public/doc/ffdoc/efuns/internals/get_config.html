get_config(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 get_config() - 查询一些驱动程序 (driver) 的系统设定值 (config settings)
　语法:
    　 string | int get_config( int );

    　 字符串 或 整数 get_config( 整数 );

　用法:
    　 这个外部函数 (efun) 用来查询一些驱动程序的系统设定值. 请参考 runtime_config.h 以了解目前有哪些可以使用的选择项目.
　参考:
    　 /include/runtime_config.h
　翻译:
    　 Spock @ FF 97.Feb.14.　 (printed 3/16/95)
