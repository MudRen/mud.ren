set_malloc_mask(3) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 set_malloc_mask() - 设定控制内存配置的除错数据显示的遮罩 (the mask controlling display of malloc debug info).
　语法:
    　 void set_malloc_mask( int mask );

    　 无返回值 set_malloc_mask( 整数 mask );

　用法:
    　 这个外部函数 (efun) 只有在编译 mud　 驱动程序 (driver) 时, options.h 中 DEBUGMALLOC　 和 DEBUGMALLOC_EXTENSIONS 皆同时定义时才有效. 此遮罩 (mask) 控制驱动程序配置内存和释放内存时, 显示多少与内存相关的除错消息. 详细数据请参阅驱动程序原始码中的 md.c
　参考:
    　 debugmalloc(3)
　翻译:
