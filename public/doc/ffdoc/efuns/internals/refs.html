refs(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 refs - 返回一个数据结构的参考数字.
　语法:
    　 int refs( mixed data );

    　 整数 refs( 混合 data );

　用法:
    　 refs() 会返回 data 的参考数字. 这个函数可以在返回 data 以前, 判断是否要复制一份数据结构.
　参考:
    　 children(3), inherit_list(3), deep_inherit_list(3), objects(3)
　翻译:
    　 Spock @ FF 97.May.24.　 (printed 3/16/95)
