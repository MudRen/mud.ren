each(3)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 each() - 分次返回一个映射变量 (mapping) 的各个 (key, value) 的内容.
　语法:
    　 mixed *each( mapping m, int reset );
　用法:
    　 each()　 函数把一个映射变量里的每对 (key, value) 一次一次地返回. each() 返回来的次序与 keys() 和 values() 两个函数返回来的次序一样. 当 each() 读到映射变量的尾端时, 会返回一个空的向量 (null vector, 即 (0,0)). 而在返回空向量之后, 下一个 each() 返回的内容又会从头开始.

    　 当不同的物件使用 each() 读取相同的映射变量时, 一定都会从头开始. 虽然 MudOS 有单执行绪 (single thread) 的性质, 但是也不致于让 each() 执行时出现问题. ( 实际上一个物件无法于使用 each() 读取映射函数读到一半时, 跳开目前读 (key, value) 的位置)

    　 如果指定第二个参数为 1, 则 each() 会重新从头开始读取.
     

　 范例:
     
      　 mixed *pair;

      　 while ((pair = each(x)) != ({})) {
      　　 write("key　　 = " + pair[0] + "\n");
      　　 write("value = " + pair[1] + "\n");
      　 }

　参考:
    　 keys(3), values(3)
