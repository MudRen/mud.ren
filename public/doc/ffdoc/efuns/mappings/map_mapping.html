map_mapping(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 map_mapping() - 经由一个函数修改一个映射变量中的元素
　语法:
    　 mapping map_mapping( mapping map, string fun, object ob, mixed extra, ... );
    　 mapping map_mapping( mapping map, function f, mixed extra, ... );

    　 映射 map_mapping( 映射 map, 字符串 fun, 物件 ob, 混合 extra, ... );
    　 映射 map_mapping( 映射 map, 函数指针 f, 混合 extra, ... );

　用法:
    　 经由 ob->fun()　 或 f　 处理一个映射变量, 并返回一个有相同关键字的新映射变量. 每一个在 map　 中的元素都会调用函数一次, 并将函数的返回值放在该项关键字之后.
　参考:
    　 filter_array(3), sort_array(3), map(3)
　翻译:
    　 Spock @ FF 97.Feb.18.　 (printed 3/16/95)
