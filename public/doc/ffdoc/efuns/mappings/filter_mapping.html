filter_mapping(3)　　 MudOS v21c2　　　 (5 Sep 1994)

　名称:
    　 filter_mapping() - 以一个函数为准, 移除一个映射变量中的某些元素.
　语法:
    　 mapping filter_mapping( mapping map, string fun, object ob, mixed extra, ... );
    　 mapping map_mapping( mapping map, function f, mixed extra, ... );

    　 映射 filter_mapping( 映射 map, 字符串 fun, 物件 ob, 混合 extra, ... );
    　 映射 filter_mapping( 映射 map, 函数 f, 混合 extra, ... );

　用法:
    　 返回一个与 map　 有相同关键字 (keys) 的映射变量, 而 map 中的所有项目 (item) 都会经过 ob->fun()　 或 f 的过滤. 每一个在 map　 中的元素都会调用 fun　 或 f 一次, 而函数 fun 或 f 的返回值会放在每一个项目之后.
　参数:
    　 filter_array(3), sort_array(3), map(3)
　翻译:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
