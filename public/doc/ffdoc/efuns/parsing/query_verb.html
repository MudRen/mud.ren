query_verb(3)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 query_verb() - 返回现在执行的命令名称.
　语法:
    　 string query_verb( void );

    　 字符串 query_verb( 无参数 );

　用法:
    　 返回目前执行的命令名称. 如果现在没有执行任何命令, 则返回 0. 这个函数的用途是: 当好几个命令会执行同一个函数时, 判断到底是哪一个命令调用此函数.
　参考:
    　 add_action(3)
　翻译:
