    目前大致上已将 MudOS v21c2 之原文文件翻译完成。不过，因为译者所学不多, 难免有遗珠之憾。未能翻译之文件清单列于附注一。

    这份文件中的分类结构如下:
    　

    • applies ―― 驱动程序对各个物件调用的函数
        • general ―― 对一般物件调用的函数
        • interactive ―― 对玩家物件调用的函数
        • master ―― 对主控物件调用的函数
    • concepts ―― 撰写 LPC 所必备的一些概念
    • lpc ―― 有关于 LPC 语法的资料
        • constructs ―― LPC 程序的保留字
        • preprocessor ―― LPC 程序的预处理命令
        • types ―― LPC 的变量、函数的类型定义
    • efuns ―― 各个外部函数的使用手册
        • arrays ―― 数组
        • buffers ―― 缓冲区
        • calls ―― 调用
        • compile ―― 编译
        • ed ―― ed 编辑程序
        • filesystem ―― 文件系统
        • floats ―― 浮点数
        • functions ―― 函数
        • general ―― 一般
        • interactive ―― 互动物件
        • internals ―― 内部
        • mappings ―― 映射
        • mudlib ―― 函数库
        • numbers ―― 数字
        • objects ―― 物件
        • parsing ―― 语法分析
        • sockets ―― 套接字
        • string ―― 字符串
        • system ―― 系统
