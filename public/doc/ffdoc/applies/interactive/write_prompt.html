write_prompt(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 write_prompt - 当命令分析器 (parser) 想显示提示符号 (prompt)　 时, 就调用此函数.
　语法:
    　 void write_prompt( void );

    　 无返回值 write_prompt( 无参数 );

　用法:
    　 如果在玩家物件中有 write_prompt(),　 则预设的提示符号该显示时, 驱动程序将调用此函数. 当玩家正处于 input_to (输入文字) 和 ed (编辑程序)　 状态时, 驱动程序不会调用 write_prompt()
　作者:
    　 Truilkan@TMI
　翻译:
