terminal_type(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 terminal_type - 告知 mudlib 使用者的终端机类型 (terminal type).
　语法:
    　 void terminal_type( string term );

    　 无返回值 terminal_type( 字符串 term );

　用法:
    　 为使用者设定终端机类型时, 可互动的物件会调用此函数. 此类型与 telnet 交涉 (negotiation)　 时所报告的相同. 如果使用者端的程序没有回应 (例如此程序不是 telnet), 则永远不再调用此函数.
　翻译:
    　 Spock @ FF 97.Aug.5.　 (printed 3/16/95)
