receive_message(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 receive_message - 提供 message() 外部函数 (efun) 的介面 (interface).
　语法:
    　 void receive_message( string class, string message );

    　 无返回值 receive_message( 字符串 class, 字符串 message );

　用法:
    　 message() 这一类的外部函数调用在玩家物件中的 receive_message().　 参数 parameter　 通常用于指示消息的种类 (说话 say, 倾诉 tell,　 表情 emote, 战斗 combat, 房间语句 room description).　 receive_message()　 和 message() 的外部函数结合在一起, 可以提供一个「聪明的」消息处理程序介面.
　参考:
    　 catch_tell(4), message(3), receive(3), receive_snoop(4)
　翻译:
    　 Spock @ FF 97.Aug.5.　 (printed 3/16/95)
