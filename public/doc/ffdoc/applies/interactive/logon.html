logon(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 logon - 初始化 (initialize) 一个连线过程 (login connection).
　语法:
    　 object logon( void );

    　 物件 logon( 无参数 );

　用法:
    　 主控物件 (master) 中的 connect() 函数返回的物件会调用 logon(). 此函数将新的使用者设定好.
　参考:
    　 connect(4)
　翻译:
    　 Spock @ FF 97.Aug.5.　 (printed 3/16/95)
