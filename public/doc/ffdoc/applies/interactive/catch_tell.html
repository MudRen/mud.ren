catch_tell(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 catch_tell - 提供一个与使用者互动的介面 (interface).
　语法:
    　 void catch_tell( string message );

    　 无返回值 catch_tell( 字符串 message );

　用法:
    　 如果编译驱动程序时, 于 options.h　 中定义了 INTERACTIVE_CATCH_TELL, 无论驱动程序对一个物件送出任何消息 ( say(), shout(), tell_object()　 等等函数) , 都会调用 catch_tell() 处理这些消息. 消息可以依照需要经由 catch_tell() 显示、丢弃、修改. catch_tell() 可以当作耳罩挡住某些消息, 也可以用于消息处理程序.
　参考:
    　 message(3), receive(3), receive_message(4)
　翻译:
    　 Spock @ FF 97.Aug.5.　 (printed 3/16/95)
