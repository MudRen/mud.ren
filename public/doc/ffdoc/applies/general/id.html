id(4)　　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 id - 此函数被 present() 调用, 以鉴定一个物件的名称.
　语法:
    　 int id( string an_id );

    　 整数 id( 字符串 an_id );

　用法:
    　 present(3)　 外部函数 (efun) 调用 id() 以判断一个指定的物件是否以字符串 an_id　 为识别名称 (id) . 如果 an_id　 是此物件的识别名称之一, 就返回 1, 不是则返回 0.
　参考:
    　 present(3)
　翻译:
    　 Spock @ FF 97.Aug.5.　 (printed 3/16/95)
