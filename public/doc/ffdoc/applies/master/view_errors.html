view_errors(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 view_errors - 检查一个使用者是否能看到错误消息.
　语法:
    　 int view_errors( object user );

    　 整数 view_error( 物件 user );

　用法:
    　 如果编译驱动程序时, options.h 中关闭了 MUDLIB_ERROR_HANDLER　 选项, 一旦使用者遇到了错误, 就会调用 view_errors( user ). 如果此函数返回非零值, 则使用者就会收到一份详细的错误消息. 反之, 则会印出预设的错误消息.　 (译按: 通常是「程序发生错误, 请通知巫师. 」一类的消息)
　参考:
    　 MUDLIB_ERROR_HANDLER(3)
