valid_shadow(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_shadow - 控制哪些物件可以作为投影 (shadow).
　语法:
    　 int valid_shadow( object ob );

    　 整数 valid_shadow( 物件 ob );

　用法:
    　 当一个物件尝试作为 ob 的投影时 (使用 shadow() 外部函数 (efun) ) , 会调用主控物件中的 valid_shadow() 函数. 物件 ob 为此函数的参数, ob　 是 previous_object()　 尝试投影的物件. 如果不准投影 ob, valid_shadow() 应返回 0 , 导致 shadow() 的调用失败并返回 0. 如果 valid_shadow()　 返回 1, 则投影成功.
　参考:
    　 shadow(3), query_shadowing(3)
　翻译:
