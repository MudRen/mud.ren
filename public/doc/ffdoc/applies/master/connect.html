connect(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 connect - 为新的使用者取得一个物件.
　语法:
    　 object connect( void );

    　 物件 connect( 无参数 );

　用法:
    　 当一个新使用者与驱动程序连线时, 驱动程序会调用主控物件 (master object) 中的 connect(). connect() 返回的物件, 其作用在于当作初始使用者物件 (initial user object). 请注意, 可以使用 exec(3)　 函数将使用者连线的物件从初始使用者物件转换到其他的物件上.
　参考:
    　 exec(3), logon(4)
　翻译:
    　 Spock @ FF 97.Aug.6.　 (printed 3/16/95)
