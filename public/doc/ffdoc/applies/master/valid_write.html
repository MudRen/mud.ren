valid_write(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_write - 检查一个人是否有权限写入一个文件.
　语法:
    　 int valid_write( string file, mixed user, string func );

    　 整数 valid_write( 字符串 file, 混合 user, 字符串 func );

　用法:
    　 每次当一个使用者尝试写入文件时, 驱动程序会调用主控物
    　 件的 valid_write()　 以检查此人的写入权限. 此函数的参
    　 数 file 为文件名称, user　 为尝试写入此档的人名, func
    　 为造成调用 valid_write()　 的函数名称. 如果
    　 valid_write() 返回非零值, 则准许写入.
　参考:
    　 valid_read(4)
　翻译:
    　 Spock @ FF 97.Aug.7.　 (printed 3/16/95)
