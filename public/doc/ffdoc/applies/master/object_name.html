object_name(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 object_name - 驱动程序调用此函数以知晓一个物件的名称.
　语法:
    　 string object_name( object );

    　 字符串 object_name( 物件 );

　用法:
    　 当 sprintf(3) 函数要印出一个物件的「值」时, 会调用此函数. 此函数应返回一个字符串, 此字符串对应于物件 object 的名称 (如一个使用者的名称).
　参考:
    　 file_name(3)
　翻译:
    　 Spock @ FF 97.Aug.6.　 (printed 3/16/95)
