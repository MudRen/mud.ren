valid_compile_to_c(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_compile_to_c - 控制一个物件在执行时段 (runtime) 中, 是否可以由 LPC->C 的方式编译 (compile).
　语法:
    　 int valid_compile_to_c( string file );

    　 整数 valid_compile_to_c( 字符串 file );

　用法:
    　 当驱动程序编译时, 如果定义了 LPC_TO_C 和 RUNTIME_LOADING,　 则载入以「.C」结尾的程序 (不是「.c」) 时, 将调用主控物件 (master object)　 的 valid_compile_to_c() 函数. 如果此函数返回 1, 此程序就编译为 C　 程序. 如果返回 0 , 则编译为普通的 LPC　 物件. 参数中的文件名称包括结尾的「.C」.
　翻译:
