valid_read(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_read - 检查一个人是否有权限读取一个文件.
　语法:
    　 int valid_read( string file, mixed user, string func );

    　 整数 valid_read( 字符串 file, 混合 user, 字符串 func );

　用法:
    　 每次当一个使用者尝试读取文件时, 驱动程序会调用主控物件的 valid_read() 以检查此人的读取权限. 此函数的参数 file　 为文件名称, user　 为尝试读取此档的人名, func 为造成调用 valid_read() 的函数名称. 如果 valid_read() 返回非零值, 则准许读取.
　参考:
    　 valid_write(4)
　翻译:
