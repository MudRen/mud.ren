author_file(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 author_file - 判断一个指定物件的作者为谁.
　语法:
    　 string author_file (string file);

    　 字符串 author_file( 字符串 file );

　用法:
    　 此函数必须存在于主控物件中 (master object). 驱动程序的作者统计函数会调用此函数, 以判断指定之物件与哪个作者有关. 此函数的判断非常武断, 其依据为该 mudlib 设计者在 author_file()　 所写的内容. 在此应注意的是, 作者所属的物件, 其一举一动 (错误、心跳数、价值等等) 将会加入作者的统计数据.
　参考:
    　 author_stats(3), domain_stats(3), domain_file(4)
　翻译:
    　 Spock @ FF 97.Aug.5.　 (printed 3/16/95)
