valid_save_binary(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_save_binary - 控制一个物件是否可以储存它的源代码.
　语法:
    　 int valid_save_binary( string file );

    　 整数 valid_save_binary( 字符串 file );

　用法:
    　 驱动程序编译时, 如果定义了 ALWAYS_SAVE_BINARIES; 或是一个物件中定义了 #pragma save_binary,　 就会以文件名称 作为参数调用 valid_save_binary().　 如果此函数返回 1, 则准许储存. 反之就不准储存.
　翻译:
    　 Spock @ FF 97.Aug.7.　 (printed 3/16/95)
