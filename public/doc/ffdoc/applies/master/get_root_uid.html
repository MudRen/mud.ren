get_root_uid(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 get_root_uid - 取得 root 使用者识别名称.
　语法:
    　 string get_root_uid( void );

    　 字符串 get_root_uid( 无参数 );

　用法:
    　 驱动程序每次载入主控物件 (master object)　 时, 会调用此函数以检验主控物件是否已经载入了, 并取得该 mud　 所自行定义的 root 使用者识别名称. 此函数必须返回一个字符串, 譬如 ROOT 定义的字符串.
　参考:
    　 get_bb_uid(4)
　翻译:
    　 Spock @ FF 97.Aug.6.　 (printed 3/16/95)
