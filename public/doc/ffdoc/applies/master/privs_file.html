privs_file(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 privs_file - 为新创造的物件指定一个私有字符串 (privs string).
　语法:
    　 string privs_file( string filename );

    　 字符串 privs_file( 字符串 filename );

　用法:
    　 当新文件创造出来时, 会调用主控物件 (master object) 中的 privs_file() 函数. 物件的文件名称为其参数, 而
    　 privs_file()　 返回的字符串就用作此新物件的私有字符串.

    　 只有在驱动程序编译时定义 PRIVS　 选项, privs_file() 才有作用.

　参考:
    　 query_privs(3), set_privs(3)
　翻译:
    　 Spock @ FF 97.Aug.6.　 (printed 3/16/95)
