crash(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 crash - 当驱动程序异常终止 (crash) 时, 就调用主控物件 (master.c)　 中的此函数.
　语法:
    　 void crash( string crash_message, object command_giver, object current_object );

    　 无返回值 crash( 字符串 crash_message, 物件 command_giver, 物件 current_object );

　用法:
    　 当驱动程序异常终止 (crash)　 时 (节段错误 segmentation fault,　 汇流排错误 bus error　 等等严重的错误. 译按: 最常见的原因是用 kill 指令终止驱动程序执行, 包括主机重新开机的情形. 至于一些原因不明的错误, 也是常有的事), 驱动程序此时会调用主控物件的 crash()　 函数. crash() 让 mudlib 可以在驱动程序终止之前将玩家和其他重要数据储存起来. crash() 也可以让你纪录一些有用的数据, 像是驱动程序如何终止的信号 (signal) 、正执行哪些物件、当时有哪些玩家等等.
　参考:
    　 slow_shutdown(4), shutdown(3)
　翻译:
