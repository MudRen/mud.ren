get_save_file_name(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 get_save_file_name - 于使用者不正常断线时, 备份其修改过的内容.
　语法:
    　 string get_save_file_name( string );

    　 字符串 get_save_file_name( 字符串 );

　用法:
    　 当一个玩家于编辑程序中编辑文件时断线的话, ed()　 函数会调用 get_save_file_name().　 此函数应返回与原来文件不同的名称, 以免覆写原来的文件.
    　 注意:
    　 此函数以前名为 get_ed_buffer_save_file_name().
　参考:
    　 ed(3)
　翻译:
    　 Spock @ FF 97.Aug.6.　 (printed 3/16/95)
