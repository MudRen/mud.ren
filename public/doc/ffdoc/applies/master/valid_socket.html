valid_socket(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_socket - 保护 socket 外部函数 (efunctions).
　语法:
    　 int valid_socket( object caller, string function, mixed *info );

    　 整数 valid_socket( 物件 caller, 字符串 function, 混合 *info );

　用法:
    　 每一个 socket 外部函数执行之前, 都会先调用 valid_socket(). 如果 valid_socket() 返回 0, 则此 socket 外部函数失败. 反之, 返回 1　 则成功. 第一个参数 caller 是调用此 socket 外部函数的物件. 第二个参数 function 是此 socket 外部函数的名称 (例如 socket_write()　 或 socket_bind() ). 第三个参数是一个信息的数组. 如果有信息的话, 这个数组的第一个元素 (element) 是参考的文件语句值 (file descriptor being referenced).　 第二个元素是拥有此 socket 的物件 (物件类型).　 第三个元素是此 socket 的远端位址 (address of the remote end)　 (字符串类型).　 第四个元素是与 socket 相关的 (associated) 连接埠号码 (port number).
