valid_link(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_link - 控制使用 link(3) 函数的情形.
　语法:
    　 int valid_link( string from, string to );

    　 整数 valid_link( 字符串 from, 字符串 to );

　用法:
    　 在 link( from, to ) 外部函数 (efun) 中, 会调用主控物件 (master object) 里面的 valid_link( from, to ) 函数. 如果 valid_link() 返回 0, 则 link() 的动作失败. 如果返回 1, 还需要以同样的参数调用 rename() 成功, 此次的 link()　 才算完成.
　参考:
    　 link(3)
　翻译:
    　 Spock @ FF 97.Aug.7.　 (printed 3/16/95)
