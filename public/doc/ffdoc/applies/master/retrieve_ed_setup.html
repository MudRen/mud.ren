retrieve_ed_setup(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 retrieve_ed_setup - 取得使用者的编辑程序设定 (setup) 或组态设定 (configuration settings).
　语法:
    　 int retrieve_ed_setup( object user );

    　 整数 retrieve_ed_setup( 物件 user );

　用法:
    　 ed()　 调用此函数, 以取得一个使用者的编辑程序设定／组态设定. 此函数应返回设定值 (设定于一个整数中).
　参考:
    　 save_ed_setup(4)
　翻译:
    　 Spock @ FF 97.Aug.6.　 (printed 3/16/95)
