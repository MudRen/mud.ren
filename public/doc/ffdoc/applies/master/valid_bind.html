valid_bind(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_bind - 控制 bind() 函数的用途.
　语法:
    　 int valid_bind( object binder, object old_owner, object new_owner );

    　 整数 valid_bind( 物件 binder, 物件 old_owner, 物件 new_owner );

　用法:
    　 当一个物件想使用 bind() 函数把某个物件的函数指针指向其他物件时, 会调用此函数检查. 如果 valid_bind() 返回 1 就允许 bind() 的动作, 返回 0　 则产生错误.
　翻译:
