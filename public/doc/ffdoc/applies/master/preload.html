preload(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 preload - 将一个物件预先载入内存.
　语法:
    　 string *epilog( int load_empty );
    　 void preload( string filename );

    　 字符串 *epilog( 整数 load_empty );
    　 无返回值 preload( 字符串 filename );

　用法:
    　 驱动程序于载入主控物件 (master object)　 之后, 会调用 epilog()　 函数. mudlib　 通常用 epilog() 初始化主控物件的数据结构 (例如系统安全表等等). epilog() 应返回一个文件名称的数组, 此数组的内容为启动游戏前, mudlib 预先想载入的物件表. 也就是说, 这些是玩家连线之前必须载入的文件. 对于数组中的每一个文件名称, 驱动程序都会为它调用主控物件中的 preload( filename ).

    　 如果启动驱动程序时加上 -e 的选项, load_empty　 参数就是非零值. 这个参数可以给 mudlib 一个信号, 决定要不要载入某些文件, 例如城堡等等.
