valid_object(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_object - 控制是否载入某个物件.
　语法:
    　 int valid_object( object obj );

    　 整数 valid_object( 物件 obj );

　用法:
    　 载入一个物件之后, 驱动程序会以新创造的此物件为参数, 调用主控物件 (master object)　 的 valid_object(). 如果 valid_object() 存在, 并返回 0, 则会摧毁此物件并使载入物件的外部函数 (efun) 产生错误输出. 如果此函数不存在或是返回 1, 则载入物件的过程就跟平常一样. 在模拟外部函数 (simul_efun) 中使用 nomask 和 inherit() 外部函数, 再与 valid_object() 并用的话, 可以安全限制 destruct() 函数.
　参考:
    　 valid_override(4)
