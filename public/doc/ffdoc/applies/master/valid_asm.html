valid_asm(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_asm - 控制 LPC->C 编译的物件是否能使用 asm { }
　语法:
    　 int valid_asm( string file );

    　 整数 valid_asm( 字符串 file );

　用法:
    　 编译驱动程序时如果开启 LPC_TO_C 选项, 则 LPC　 源代码中一旦出现 asm { }　 结构就会调用 valid_asm()　 函数. 如果此函数返回 0, 就中断编译过程并产生错误.

    　 asm { } 结构的用法如下:

    　 asm {
    　　 < C 源代码 >
    　 }

    　 如此一来, 可以在 asm { } 之间插入 C　 源代码.

　参考:
    　 valid_compile_to_c(4)
　翻译:
