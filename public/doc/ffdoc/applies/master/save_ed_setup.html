save_ed_setup(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 save_ed_setup - 储存一个使用者的编辑程序设定或组态设定.
　语法:
    　 int save_ed_setup( object user, int config );

    　 整数 save_ed_setup( 物件 user, 整数 config );

　用法:
    　 ed()　 调用此函数以储存一个使用者的编辑程序设定／组态设定 (存于一个整数中). 储存成功时, 此函数应返回 1　 或「真」值 (译按: 即非零值). 反之则返回 0　 或「伪」值 (译按: 即零值).
　参考:
    　 retrieve_ed_setup(4)
　翻译:
    　 Spock @ FF 97.Aug.5.　 (printed 3/16/95)
