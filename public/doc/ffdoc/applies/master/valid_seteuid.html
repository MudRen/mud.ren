valid_seteuid(4) MudOS v21c2　 (5 Sep 1994)

　名称:
    　 valid_seteuid - 保障使用 seteuid(3) 的系统安全.
　语法:
    　 int valid_seteuid( object obj, string euid );

    　 整数 valid_seteuid( 物件 obj, 字符串 euid );

　用法:
    　 在 seteuid( euid ) 函数里面, 驱动程序调用主控物件 (master object)　 中的 valid_seteuid( ob, euid ) 函数. 如果 valid_seteuid() 返回 0, 则 seteuid(3) 失败. 如果 valid_seteuid() 返回 1, 则 seteuid(3) 成功.
　参考:
    　 seteuid(3), geteuid(3), getuid(3), export_uid(3)
　翻译:
    　 Spock @ FF 97.Aug.7.　 (printed 3/16/95)
