log_error(4)　 MudOS v21c2　 (5 Sep 1994)

　名称:
    　 log_error - 聪明地纪录错误.
　语法:
    　 void log_error( string file, string message );

    　 无返回值 log_error( 字符串 file, 字符串 message );

　用法:
    　 无论编译程序 (compilation)　 时发生任何错误, 驱动程序都会以发生错误的文件名称为 file 参数、错误消息为 message 参数, 调用主控物件 (master object)　 中的 log_error() 函数. 调用此函数时, log_error() 可以做任何您认为应该做的数据纪录. 通常 log_error()　 根据发生错误的文件名称, 决定该纪录要写入哪个错误消息文件中.
　参考:
    　 error_handler(4)
